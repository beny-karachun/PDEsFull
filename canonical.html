Canonical Forms of Second-Order Linear PDEs

Theory Focus

Consider the general second-order linear PDE:

$$ Au_{xx} + Bu_{xy} + Cu_{yy} + Du_x + Eu_y + Fu = G $$

Where $A, B, C, D, E, F, G$ can be functions of $x$ and $y$. The classification and canonical form depend only on the principal part ($A, B, C$).

1. Classification: Calculate the discriminant $\Delta = B^2 - 4AC$.

  • $\Delta > 0$: Hyperbolic
  • $\Delta = 0$: Parabolic
  • $\Delta < 0$: Elliptic

2. Characteristic Equation: To find the transformation coordinates $(\xi, \eta)$ that simplify the PDE, we solve the characteristic equation for $dy/dx$: $$ A \left(\frac{dy}{dx}\right)^2 - B \left(\frac{dy}{dx}\right) + C = 0 $$ This is a quadratic equation for $dy/dx$. Solving it gives the slopes of the characteristic curves. Integrating $dy/dx = \dots$ gives equations of the form $\phi(x,y) = \text{constant}$. These $\phi(x,y)$ functions are used as the new coordinates.

3. Canonical Forms (via coordinate transformation $\xi=\phi_1(x,y), \eta=\phi_2(x,y)$):

Hyperbolic Case ($\Delta > 0$)

  • The characteristic equation has two distinct real roots for $dy/dx$.
  • Integrating gives two distinct families of real characteristic curves: $\phi_1(x,y) = C_1$ and $\phi_2(x,y) = C_2$.
  • Choose the transformation $\xi = \phi_1(x,y)$ and $\eta = \phi_2(x,y)$.
  • The PDE transforms into the first canonical form: $$ u_{\xi\eta} + \text{(lower order terms in } u_\xi, u_\eta, u) = \tilde{G} $$
  • Alternatively, a further linear transformation $\alpha = \xi+\eta, \beta = \xi-\eta$ leads to the second canonical form: $$ u_{\alpha\alpha} - u_{\beta\beta} + \text{(lower order terms)} = \tilde{\tilde{G}} $$ (This resembles the standard wave equation form).

Parabolic Case ($\Delta = 0$)

  • The characteristic equation has one real repeated root for $dy/dx$.
  • Integrating gives one family of real characteristic curves: $\phi(x,y) = C_1$.
  • Choose the transformation $\xi = \phi(x,y)$. For the second coordinate, choose $\eta$ to be *any* function independent of $\xi$. Often $\eta=x$ or $\eta=y$ works conveniently.
  • The PDE transforms into the canonical form: $$ u_{\eta\eta} + \text{(lower order terms)} = \tilde{G} $$ (Or possibly $u_{\xi\xi} + \dots = \tilde{G}$, depending on the choice of $\eta$).

Elliptic Case ($\Delta < 0$)

  • The characteristic equation has two complex conjugate roots for $dy/dx$.
  • Integrating gives two families of complex characteristic curves $\phi(x,y) = C_1$ and $\bar{\phi}(x,y) = C_2$. Note $\phi$ will be complex.
  • Choose the transformation using the real and imaginary parts of one characteristic function: $\xi = \text{Re}(\phi(x,y))$ and $\eta = \text{Im}(\phi(x,y))$.
  • The PDE transforms into the canonical form (Laplace-like form): $$ u_{\xi\xi} + u_{\eta\eta} + \text{(lower order terms)} = \tilde{G} $$

Note: Transforming the lower-order terms ($D, E, F, G$) can be algebraically intensive and is often omitted unless specifically required. The primary goal is usually to simplify the principal part.

Practice Questions

Question 1:

Classify and find the canonical form (principal part only) for $u_{xx} - 5u_{xy} + 6u_{yy} = 0$.

Solution:

$A=1, B=-5, C=6$. $\Delta = B^2 - 4AC = (-5)^2 - 4(1)(6) = 25 - 24 = 1 > 0$. Hyperbolic.

Characteristic equation: $A(y')^2 - B y' + C = 0 \implies 1(y')^2 - (-5)y' + 6 = 0 \implies (y')^2 + 5y' + 6 = 0$.

$(y'+2)(y'+3) = 0$. Roots: $y'=-2$ and $y'=-3$.

Integrate: $dy/dx = -2 \implies y = -2x + C_1 \implies y+2x = C_1$. Let $\xi = y+2x$.

Integrate: $dy/dx = -3 \implies y = -3x + C_2 \implies y+3x = C_2$. Let $\eta = y+3x$.

Canonical form (principal part): $u_{\xi\eta} = 0$ (plus lower order terms, which are zero in this simple case after transformation).

Question 2:

Classify and find the canonical form (principal part only) for $u_{xx} + 4u_{xy} + 4u_{yy} = 0$.

Solution:

$A=1, B=4, C=4$. $\Delta = B^2 - 4AC = 4^2 - 4(1)(4) = 16 - 16 = 0$. Parabolic.

Characteristic equation: $A(y')^2 - B y' + C = 0 \implies 1(y')^2 - 4y' + 4 = 0 \implies (y'-2)^2 = 0$.

Repeated root: $y'=2$.

Integrate: $dy/dx = 2 \implies y = 2x + C_1 \implies y-2x = C_1$. Let $\xi = y-2x$.

Choose $\eta$ independent, e.g., $\eta = x$.

Canonical form (principal part): $u_{\eta\eta} = 0$ (plus lower order terms).

Question 3:

Classify and find the canonical form (principal part only) for $u_{xx} + u_{xy} + u_{yy} = 0$.

Solution:

$A=1, B=1, C=1$. $\Delta = B^2 - 4AC = 1^2 - 4(1)(1) = 1 - 4 = -3 < 0$. Elliptic.

Characteristic equation: $A(y')^2 - B y' + C = 0 \implies (y')^2 - y' + 1 = 0$.

Roots: $y' = \frac{-(-1) \pm \sqrt{(-1)^2 - 4(1)(1)}}{2(1)} = \frac{1 \pm \sqrt{1-4}}{2} = \frac{1 \pm i\sqrt{3}}{2}$.

Integrate one root, e.g., $dy/dx = \frac{1 + i\sqrt{3}}{2}$. $dy = (\frac{1 + i\sqrt{3}}{2}) dx$.

$y = \frac{1 + i\sqrt{3}}{2} x + C$. So $y - \frac{1}{2}x = i\frac{\sqrt{3}}{2} x + C$. $\frac{2y-x}{2} = i\frac{\sqrt{3}}{2} x + C$.

$2y-x - i\sqrt{3}x = C' = \phi(x,y)$.

Choose real and imaginary parts: $\xi = \text{Re}(\phi) = 2y-x$. $\eta = \text{Im}(\phi) = -\sqrt{3}x$. (We can scale $\eta$ for convenience, let $\eta = \sqrt{3}x$).

Canonical form (principal part): $u_{\xi\xi} + u_{\eta\eta} = 0$ (plus lower order terms).

Question 4:

Classify $y u_{xx} + x u_{yy} = 0$ for $x>0, y>0$.

Solution:

$A=y, B=0, C=x$.

$\Delta = B^2 - 4AC = 0^2 - 4(y)(x) = -4xy$.

Since $x>0$ and $y>0$, $xy>0$. Therefore $\Delta = -4xy < 0$.

The PDE is Elliptic in this region.

Question 5:

Find the characteristic curves for $u_{tt} - c^2 u_{xx} = 0$.

Solution:

Let $y \to t$. PDE: $u_{tt} - c^2 u_{xx} = 0$. $A=-c^2, B=0, C=1$.

Characteristic equation: $A(dt/dx)^2 - B(dt/dx) + C = 0$.

$-c^2 (dt/dx)^2 - 0 + 1 = 0 \implies (dt/dx)^2 = 1/c^2 \implies dt/dx = \pm 1/c$.

Integrate $dt/dx = 1/c \implies t = x/c + K_1 \implies x - ct = C_1$.

Integrate $dt/dx = -1/c \implies t = -x/c + K_2 \implies x + ct = C_2$.

The characteristic curves are the lines $x-ct = \text{constant}$ and $x+ct = \text{constant}$.

Question 6:

Classify and find the transformation to canonical form for $u_{xx} + 2u_{xy} + u_{yy} = 0$.

Solution:

$A=1, B=2, C=1$. $\Delta = 2^2 - 4(1)(1) = 0$. Parabolic.

Characteristic equation: $1(y')^2 - 2y' + 1 = 0 \implies (y'-1)^2 = 0$.

Repeated root $y'=1$.

Integrate: $dy/dx = 1 \implies y = x + C_1 \implies y-x = C_1$. Let $\xi = y-x$.

Choose $\eta$ independent, e.g., $\eta = x$.

The transformation is $\xi=y-x, \eta=x$. Canonical form: $u_{\eta\eta}=0$.

Question 7:

Classify and find the characteristic curves for $4u_{xx} - u_{yy} = 0$.

Solution:

Let $x \to x, y \to y$. $A=4, B=0, C=-1$. $\Delta = 0^2 - 4(4)(-1) = 16 > 0$. Hyperbolic.

Characteristic equation: $A(y')^2 - B y' + C = 0 \implies 4(y')^2 - 0 - 1 = 0$.

$(y')^2 = 1/4 \implies y' = \pm 1/2$.

Integrate $dy/dx = 1/2 \implies y = x/2 + C_1 \implies 2y - x = C_1'$. Let $\phi_1 = 2y-x$.

Integrate $dy/dx = -1/2 \implies y = -x/2 + C_2 \implies 2y + x = C_2'$. Let $\phi_2 = 2y+x$.

Characteristic curves: $2y-x = \text{constant}$ and $2y+x = \text{constant}$.

Question 8:

Classify $x^2 u_{xx} + 2xy u_{xy} + y^2 u_{yy} = 0$.

Solution:

$A=x^2, B=2xy, C=y^2$.

$\Delta = B^2 - 4AC = (2xy)^2 - 4(x^2)(y^2) = 4x^2y^2 - 4x^2y^2 = 0$.

The PDE is Parabolic (where defined, i.e., not on axes if needed).

Question 9:

Find the canonical form (principal part) for the PDE in Question 8.

Solution:

Characteristic equation: $A(y')^2 - B y' + C = 0 \implies x^2(y')^2 - 2xy y' + y^2 = 0$.

This is $(xy' - y)^2 = 0$. So $xy' - y = 0$.

$x \frac{dy}{dx} = y \implies \frac{dy}{y} = \frac{dx}{x}$.

Integrate: $\ln|y| = \ln|x| + C_1' \implies \ln|y/x| = C_1'$. So $y/x = C_1$. Let $\xi = y/x$.

Choose $\eta$ independent, e.g., $\eta = x$.

Canonical form (principal part): $u_{\eta\eta} = 0$ (plus lower order terms).

Question 10:

Classify and find the characteristics for $u_{xx} + 5u_{xy} + 4u_{yy} = 0$.

Solution:

$A=1, B=5, C=4$. $\Delta = 5^2 - 4(1)(4) = 25 - 16 = 9 > 0$. Hyperbolic.

Characteristic equation: $1(y')^2 - 5y' + 4 = 0 \implies (y'-1)(y'-4) = 0$.

Roots $y'=1$ and $y'=4$.

Integrate $dy/dx=1 \implies y = x + C_1 \implies y-x = C_1$.

Integrate $dy/dx=4 \implies y = 4x + C_2 \implies y-4x = C_2$.

Characteristics: $y-x = \text{constant}$ and $y-4x = \text{constant}$.

Question 11:

Classify $u_{xx} + u_{yy} + u = 0$.

Solution:

$A=1, B=0, C=1$. $\Delta = 0^2 - 4(1)(1) = -4 < 0$. Elliptic.

Question 12:

Classify $u_{tt} + 2u_{tx} + u_{xx} = 0$.

Solution:

Let $y \to t$. $A=1, B=2, C=1$. $\Delta = 2^2 - 4(1)(1) = 0$. Parabolic.

Question 13:

Find the characteristic equation for $3u_{xx} + 10u_{xy} + 3u_{yy} = 0$.

Solution:

$A=3, B=10, C=3$. Characteristic equation: $A(y')^2 - B y' + C = 0$.

$3(y')^2 - 10y' + 3 = 0$.

Question 14:

Solve the characteristic equation from Q13 and classify the PDE.

Solution:

$3(y')^2 - 10y' + 3 = 0$. Using quadratic formula:

$y' = \frac{-(-10) \pm \sqrt{(-10)^2 - 4(3)(3)}}{2(3)} = \frac{10 \pm \sqrt{100 - 36}}{6} = \frac{10 \pm \sqrt{64}}{6} = \frac{10 \pm 8}{6}$.

Roots are $y' = \frac{18}{6} = 3$ and $y' = \frac{2}{6} = \frac{1}{3}$.

Since there are two distinct real roots, the PDE is Hyperbolic. (Also $\Delta = B^2-4AC = 10^2-4(3)(3)=100-36=64>0$).

Question 15:

Find the transformation coordinates $(\xi, \eta)$ for the PDE in Q13.

Solution:

From Q14, the characteristic equations are $y'=3$ and $y'=1/3$.

Integrate $dy/dx=3 \implies y=3x+C_1 \implies y-3x = C_1$. Let $\xi = y-3x$.

Integrate $dy/dx=1/3 \implies y=x/3+C_2 \implies 3y-x = C_2'$. Let $\eta = 3y-x$.

The transformation is $\xi = y-3x, \eta = 3y-x$.

Question 16:

Classify $u_{xx} - 2 u_{xy} + 2 u_{yy} + \dots = 0$.

Solution:

$A=1, B=-2, C=2$. $\Delta = (-2)^2 - 4(1)(2) = 4 - 8 = -4 < 0$. Elliptic.

Question 17:

Find the transformation $(\xi, \eta)$ for the PDE in Q16.

Solution:

Characteristic equation: $1(y')^2 - (-2)y' + 2 = 0 \implies (y')^2 + 2y' + 2 = 0$.

Roots: $y' = \frac{-2 \pm \sqrt{2^2 - 4(1)(2)}}{2(1)} = \frac{-2 \pm \sqrt{4-8}}{2} = \frac{-2 \pm \sqrt{-4}}{2} = \frac{-2 \pm 2i}{2} = -1 \pm i$.

Integrate $dy/dx = -1+i$. $y = (-1+i)x + C$. $y+x = ix + C$. $y+x - ix = C' = \phi(x,y)$.

Real part: $\xi = \text{Re}(\phi) = y+x$.

Imaginary part: $\eta = \text{Im}(\phi) = -x$. (Can choose $\eta=x$ for simplicity).

Transformation: $\xi = y+x, \eta = x$.

Question 18:

Classify $u_{xx} - y u_{yy} = 0$.

Solution:

$A=1, B=0, C=-y$.

$\Delta = B^2 - 4AC = 0^2 - 4(1)(-y) = 4y$.

  • If $y > 0$, $\Delta > 0 \implies$ Hyperbolic.
  • If $y = 0$, $\Delta = 0 \implies$ Parabolic.
  • If $y < 0$, $\Delta < 0 \implies$ Elliptic.

Question 19:

Find the characteristic curves for $u_{xx} - y u_{yy} = 0$ where $y>0$.

Solution:

In the region $y>0$, the PDE is hyperbolic.

Characteristic equation: $A(y')^2 - By' + C = 0 \implies 1(y')^2 - 0 - y = 0$.

$(y')^2 = y \implies y' = \pm \sqrt{y}$.

Integrate $dy/dx = \sqrt{y} \implies y^{-1/2} dy = dx \implies 2y^{1/2} = x + C_1 \implies x - 2\sqrt{y} = C_1'$.

Integrate $dy/dx = -\sqrt{y} \implies y^{-1/2} dy = -dx \implies 2y^{1/2} = -x + C_2 \implies x + 2\sqrt{y} = C_2$.

Characteristic curves: $x \pm 2\sqrt{y} = \text{constant}$.

Question 20:

Classify $u_{tt} - u_{xx} - u_{yy} = 0$ (2D Wave Equation).

Solution:

Classification is typically defined for PDEs in two independent variables. This PDE has three ($t, x, y$). While related concepts exist, the $B^2-4AC$ classification doesn't directly apply in its standard form.

However, if we fix one variable or consider characteristics in higher dimensions, it exhibits hyperbolic behavior. For the purpose of the standard classification test, this problem is outside its scope.

(Often considered Hyperbolic based on its wave-like solutions).

Question 21:

Classify and find canonical form (principal part) for $9u_{xx} + 6u_{xy} + u_{yy} = 0$.

Solution:

$A=9, B=6, C=1$. $\Delta = 6^2 - 4(9)(1) = 36-36=0$. Parabolic.

Char. Eq: $9(y')^2 - 6y' + 1 = 0 \implies (3y'-1)^2 = 0$.

Repeated root $y'=1/3$.

Integrate: $dy/dx=1/3 \implies y=x/3+C_1 \implies 3y-x=C_1'$. Let $\xi = 3y-x$.

Choose $\eta=x$. Canonical form: $u_{\eta\eta}=0$.

Question 22:

Classify and find characteristics for $u_{xx} - u_{xy} - 2u_{yy} = 0$.

Solution:

$A=1, B=-1, C=-2$. $\Delta = (-1)^2 - 4(1)(-2) = 1+8=9>0$. Hyperbolic.

Char. Eq: $1(y')^2 - (-1)y' + (-2) = 0 \implies (y')^2 + y' - 2 = 0$.

$(y'+2)(y'-1)=0$. Roots $y'=-2$ and $y'=1$.

Integrate $y'=-2 \implies y+2x=C_1$.

Integrate $y'=1 \implies y-x=C_2$.

Characteristics: $y+2x=\text{const}, y-x=\text{const}$.

Question 23:

Classify $u_{xx} + e^x u_{yy} = 0$.

Solution:

$A=1, B=0, C=e^x$.

$\Delta = 0^2 - 4(1)(e^x) = -4e^x$.

Since $e^x$ is always positive, $\Delta < 0$. Elliptic.

Question 24:

Classify and find the canonical transformation for $u_{xx} + 2u_{xy} + 10u_{yy} = 0$.

Solution:

$A=1, B=2, C=10$. $\Delta = 2^2 - 4(1)(10) = 4-40 = -36 < 0$. Elliptic.

Char. Eq: $1(y')^2 - 2y' + 10 = 0$.

$y' = \frac{-(-2) \pm \sqrt{(-2)^2 - 4(1)(10)}}{2(1)} = \frac{2 \pm \sqrt{4-40}}{2} = \frac{2 \pm \sqrt{-36}}{2} = \frac{2 \pm 6i}{2} = 1 \pm 3i$.

Integrate $dy/dx = 1+3i$. $y = (1+3i)x + C$. $y-x = 3ix + C$. $y-x - 3ix = C' = \phi$.

Real part $\xi = y-x$. Imaginary part $\eta = -3x$. (Choose $\eta = 3x$).

Transformation: $\xi=y-x, \eta=3x$. Canonical form $u_{\xi\xi}+u_{\eta\eta}=0$.

Question 25:

Classify $u_{tt} - (t^2+1) u_{xx} = 0$ for $t>0$.

Solution:

Let $y \to t$. $A=-(t^2+1), B=0, C=1$.

$\Delta = 0^2 - 4(-(t^2+1))(1) = 4(t^2+1)$.

Since $t^2+1 > 0$ for all real $t$, $\Delta > 0$. Hyperbolic.

Question 26:

Find the characteristics for $u_{xx} + 3u_{xy} + 2u_{yy} = 0$.

Solution:

$A=1, B=3, C=2$. $\Delta = 3^2 - 4(1)(2) = 9-8=1>0$. Hyperbolic.

Char. Eq: $1(y')^2 - 3y' + 2 = 0 \implies (y'-1)(y'-2) = 0$.

Roots $y'=1, y'=2$.

Integrate $y'=1 \implies y-x=C_1$.

Integrate $y'=2 \implies y-2x=C_2$.

Characteristics: $y-x=\text{const}, y-2x=\text{const}$.

Question 27:

Classify the PDE $u_{xx} - 2u_{xy} + u_{yy} - u_x + 2u_y - u = 0$.

Solution:

$A=1, B=-2, C=1$.

$\Delta = B^2 - 4AC = (-2)^2 - 4(1)(1) = 4-4 = 0$.

Parabolic.

Question 28:

Find the transformation to canonical form for the PDE in Q27.

Solution:

Char. Eq: $1(y')^2 - (-2)y' + 1 = 0 \implies (y')^2 + 2y' + 1 = 0 \implies (y'+1)^2 = 0$.

Repeated root $y'=-1$.

Integrate $dy/dx = -1 \implies y = -x+C_1 \implies y+x = C_1$. Let $\xi = y+x$.

Choose $\eta$ independent, e.g., $\eta = x$.

Transformation $\xi=y+x, \eta=x$. Canonical form $u_{\eta\eta} + \dots = 0$.

Question 29:

Classify $u_{xx} + 4u_{yy} = 0$.

Solution:

$A=1, B=0, C=4$.

$\Delta = 0^2 - 4(1)(4) = -16 < 0$. Elliptic.

Question 30:

Find the transformation $(\xi, \eta)$ for the PDE in Q29.

Solution:

Char. Eq: $1(y')^2 - 0y' + 4 = 0 \implies (y')^2 + 4 = 0$.

$y' = \pm \sqrt{-4} = \pm 2i$.

Integrate $dy/dx = 2i$. $y = 2ix + C$. $y - 2ix = C = \phi$.

Real part $\xi = y$. Imaginary part $\eta = -2x$. (Choose $\eta = 2x$).

Transformation $\xi=y, \eta=2x$. Canonical form $u_{\xi\xi}+u_{\eta\eta}=0$.

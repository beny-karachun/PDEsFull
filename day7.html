Day 7: Mixed Review / Exam Practice

Theory Focus (Summary & Strategy)

Today focuses on identifying the correct method and applying the procedures learned throughout the week.

  • First-Order PDE ($a u_x + b u_y = c$): Use the Method of Characteristics. Set up $\frac{dx}{a} = \frac{dy}{b} = \frac{du}{c}$. Find two characteristics. Use IVP to find the arbitrary function.
  • Second-Order PDE Classification ($Au_{xx}+Bu_{xy}+Cu_{yy}+...$): Calculate $\Delta = B^2 - 4AC$. Classify as Hyperbolic ($\Delta>0$), Parabolic ($\Delta=0$), or Elliptic ($\Delta<0$).
  • Wave Equation ($u_{tt}=c^2u_{xx}$):
    • Infinite Domain (IVP): Use D'Alembert's Formula: $u(x,t) = \frac{1}{2}[f(x+ct)+f(x-ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct} g(s)ds$.
    • Finite Domain (IBVP, usually $u=0$ BCs): Use Separation of Variables. $X_n$ are sines (usually), $T_n$ involves $\cos(\omega_n t), \sin(\omega_n t)$. Need two Fourier series (for $f(x)$ and $g(x)$) to find coefficients $A_n, B_n$.
  • Heat Equation ($u_{t}=k u_{xx}$):
    • Finite Domain (IBVP, usually $u=0$ BCs): Use Separation of Variables. $X_n$ are sines (usually), $T_n$ involves $e^{-k\lambda_n t}$. Need one Fourier series (for $f(x)$) to find coefficients $B_n$.
  • Laplace Equation ($u_{xx}+u_{yy}=0$):
    • Rectangle (BVP, usually $u=0$ on 3 sides): Use Separation of Variables. One variable ($X$ or $Y$) gives trigonometric functions (eigenvalue problem), the other gives hyperbolic functions. Use superposition if needed. Need one Fourier series (for the non-homogeneous side) to find coefficients $E_n$.
  • Key Skills: Solving basic ODEs, Integration (especially by parts), calculating Fourier Sine coefficients $\frac{2}{L}\int_0^L h(x)\sin(\frac{n\pi x}{L})dx$.

Practice Questions

Question 1:

Classify the PDE $u_{xx} + 2u_{xy} + u_{yy} + u_x = 0$.

Solution:

$A=1, B=2, C=1$. $\Delta = B^2 - 4AC = 2^2 - 4(1)(1) = 0$.

The PDE is Parabolic.

Question 2:

Solve the IVP: $u_x + 3u_y = 0$, $u(0,y) = \cos(y)$.

Solution:

Method of Characteristics. $\frac{dx}{1} = \frac{dy}{3} = \frac{du}{0}$.

$u=C_1$. $3dx=dy \implies y-3x=C_2$.

General solution $u(x,y) = F(y-3x)$.

$u(0,y) = F(y-0) = F(y) = \cos(y)$.

Solution $u(x,y) = \cos(y-3x)$.

Question 3:

Solve $u_{tt} - 4u_{xx} = 0$ for $x \in (-\infty, \infty)$, with $u(x,0) = x^2$ and $u_t(x,0) = 0$.

Solution:

D'Alembert's Formula. $c=2, f(x)=x^2, g(x)=0$.

$u(x,t) = \frac{1}{2}[f(x+2t) + f(x-2t)] + 0$.

$u(x,t) = \frac{1}{2}[(x+2t)^2 + (x-2t)^2] = \frac{1}{2}[x^2+4xt+4t^2 + x^2-4xt+4t^2]$.

$u(x,t) = \frac{1}{2}[2x^2 + 8t^2] = x^2 + 4t^2$.

Question 4:

Find the eigenvalues $\lambda_n$ and eigenfunctions $X_n(x)$ for $X'' + \lambda X = 0$ with $X(0)=0, X'(1)=0$.

Solution:

$L=1$. Mixed BCs.

Case 1: $\lambda=0$. $X(x)=Ax+B$. $X(0)=B=0$. $X'(x)=A$. $X'(1)=A=0$. Trivial solution.

Case 2: $\lambda=-\mu^2<0$. $X(x)=Ae^{\mu x}+Be^{-\mu x}$. $X(0)=A+B=0 \implies B=-A$. $X(x)=A(e^{\mu x}-e^{-\mu x})$. $X'(x)=A\mu(e^{\mu x}+e^{-\mu x})$. $X'(1)=A\mu(e^{\mu}+e^{-\mu})=0$. Since $\mu\neq 0$ and $e^\mu+e^{-\mu}=2\cosh\mu \neq 0$, we need $A=0$. Trivial solution.

Case 3: $\lambda=\mu^2>0$. $X(x)=A\cos(\mu x)+B\sin(\mu x)$. $X(0)=A=0$. $X(x)=B\sin(\mu x)$. $X'(x)=B\mu\cos(\mu x)$. $X'(1)=B\mu\cos(\mu)=0$. For non-trivial $B$, need $\cos(\mu)=0$. $\mu = (n-\frac{1}{2})\pi = \frac{(2n-1)\pi}{2}$ for $n=1, 2, 3, \dots$.

Eigenvalues $\lambda_n = \mu^2 = \left(\frac{(2n-1)\pi}{2}\right)^2$.

Eigenfunctions $X_n(x) = \sin\left(\frac{(2n-1)\pi x}{2}\right)$.

Question 5:

Solve the heat equation $u_t = 2 u_{xx}$ for $x \in (0, \pi)$, with $u(0,t)=0, u(\pi,t)=0$, and $u(x,0)=3\sin(x) - \sin(4x)$.

Solution:

Separation of Variables (Heat, Dirichlet). $k=2, L=\pi$. $\lambda_n=n^2, X_n=\sin(nx)$.

$u(x,t) = \sum B_n e^{-k\lambda_n t} \sin(nx) = \sum B_n e^{-2n^2 t} \sin(nx)$.

$u(x,0) = \sum B_n \sin(nx) = 3\sin(1x) - 1\sin(4x)$.

Matching coefficients: $B_1=3, B_4=-1$. All other $B_n=0$.

$u(x,t) = B_1 e^{-2(1)^2 t}\sin(x) + B_4 e^{-2(4)^2 t}\sin(4x)$.

$u(x,t) = 3e^{-2t}\sin(x) - e^{-32t}\sin(4x)$.

Question 6:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,1), y \in (0,1)$, with $u(0,y)=0, u(1,y)=0, u(x,0)=\sin(2\pi x), u(x,1)=0$.

Solution:

Laplace Eq, Rectangle. $a=1, b=1$. Homogeneous on $x=0, x=1, y=1$. Non-homogeneous on $y=0$. $f(x)=\sin(2\pi x)$.

Use setup from Day 6, Q10.

$X_n=\sin(n\pi x)$. $Y_n(y) = E_n \sinh(n\pi(1-y))$.

$u(x,y) = \sum E_n \sinh(n\pi(1-y))\sin(n\pi x)$.

$u(x,0) = \sum E_n \sinh(n\pi)\sin(n\pi x) = \sin(2\pi x)$.

Matching coefficients: $n=2$. $E_2 \sinh(2\pi) = 1$. $E_n=0$ for $n\neq 2$.

$E_2 = 1/\sinh(2\pi)$.

$u(x,y) = E_2 \sinh(2\pi(1-y))\sin(2\pi x) = \frac{\sinh(2\pi(1-y))\sin(2\pi x)}{\sinh(2\pi)}$.

Question 7:

Classify the PDE $u_{tt} - 2u_{xt} + u_{xx} = 0$.

Solution:

Let $y \to t$. $A=1, B=-2, C=1$.

$\Delta = B^2 - 4AC = (-2)^2 - 4(1)(1) = 4-4 = 0$.

The PDE is Parabolic.

Question 8:

Find the general solution to $u_x + 2xu_y = 0$.

Solution:

Characteristics: $\frac{dx}{1} = \frac{dy}{2x} = \frac{du}{0}$.

$u=C_1$.

From $\frac{dx}{1} = \frac{dy}{2x}$, $2x dx = dy$. Integrating gives $x^2 = y + C_2$. So $y-x^2 = C_2$.

General solution $u(x,y) = F(y-x^2)$.

Question 9:

Find $u(x,t)$ if $u_{tt} - 9u_{xx} = 0$, $u(x,0) = 0$, $u_t(x,0) = \sin(x)$.

Solution:

D'Alembert. $c=3, f(x)=0, g(x)=\sin(x)$.

$u(x,t) = 0 + \frac{1}{2(3)} \int_{x-3t}^{x+3t} \sin(s) ds = \frac{1}{6} [-\cos s]_{x-3t}^{x+3t}$.

$u(x,t) = \frac{1}{6} [-\cos(x+3t) + \cos(x-3t)] = \frac{1}{6} [\cos(x-3t) - \cos(x+3t)]$.

Using $\cos A - \cos B = -2\sin(\frac{A+B}{2})\sin(\frac{A-B}{2})$:

$u(x,t) = \frac{1}{6} [-2 \sin(\frac{x-3t+x+3t}{2}) \sin(\frac{x-3t-(x+3t)}{2}) ]$.

$u(x,t) = \frac{1}{6} [-2 \sin(x) \sin(-3t)] = \frac{1}{6} [2\sin x \sin(3t)] = \frac{1}{3}\sin x \sin(3t)$.

Question 10:

Find the Fourier Sine Series coefficient $B_3$ for $f(x)=x$ on $[0, 1]$.

Solution:

$L=1$. $B_n = \frac{2}{1} \int_0^1 x \sin(n\pi x) dx$.

From Day 4 Q6 (with $L=1$), $B_n = \frac{2(1)(-1)^{n+1}}{n\pi}$.

For $n=3$, $B_3 = \frac{2(-1)^{3+1}}{3\pi} = \frac{2(-1)^4}{3\pi} = \frac{2}{3\pi}$.

Question 11:

Solve the wave equation $u_{tt} = c^2 u_{xx}$ for $x \in (0, \pi)$, $u(0,t)=u(\pi,t)=0$, $u(x,0)=0$, $u_t(x,0)=4\sin(x)\cos(x)$.

Solution:

$L=\pi$. $f(x)=0 \implies A_n=0$. $g(x)=4\sin x \cos x = 2\sin(2x)$. $\omega_n=nc$.

$g(x) = \sum (B_n \omega_n) \sin(nx) = 2\sin(2x)$.

Matching: $B_2 \omega_2 = 2$. $B_n=0$ for $n\neq 2$.

$\omega_2=2c$. $B_2(2c)=2 \implies B_2 = 1/c$.

$u(x,t) = B_2 \sin(\omega_2 t) \sin(2x) = \frac{1}{c} \sin(2ct) \sin(2x)$.

Question 12:

Classify the PDE $u_{xx} - u_{yy} + u_x + u_y = 0$.

Solution:

$A=1, B=0, C=-1$.

$\Delta = B^2 - 4AC = 0^2 - 4(1)(-1) = 4$.

Since $\Delta > 0$, the PDE is Hyperbolic.

Question 13:

Solve $u_t = u_{xx}$ for $x \in (0, 1)$, $u(0,t)=u(1,t)=0$, $u(x,0)=1$.

Solution:

$k=1, L=1$. $\lambda_n=(n\pi)^2, X_n=\sin(n\pi x)$.

$u(x,t) = \sum B_n e^{-(n\pi)^2 t} \sin(n\pi x)$.

$u(x,0) = \sum B_n \sin(n\pi x) = 1$.

$B_n$ are Fourier Sine coefficients of $f(x)=1$ on $[0,1]$.

$B_n = \frac{2}{1} \int_0^1 1 \cdot \sin(n\pi x) dx = 2 [-\frac{\cos(n\pi x)}{n\pi}]_0^1 = \frac{2(1-(-1)^n)}{n\pi}$.

$B_n=0$ for even $n$. For odd $n=2m-1$, $B_{2m-1} = 4/((2m-1)\pi)$.

$u(x,t) = \sum_{m=1}^{\infty} \frac{4}{(2m-1)\pi} e^{-((2m-1)\pi)^2 t} \sin((2m-1)\pi x)$.

Question 14:

Find $u(1,1)$ if $u_{tt} - u_{xx} = 0$, $u(x,0)=0$, $u_t(x,0)=x$.

Solution:

D'Alembert. $c=1, f(x)=0, g(x)=x$. Point $(1,1)$.

$x+ct = 1+1(1)=2$. $x-ct=1-1(1)=0$.

$u(1,1) = 0 + \frac{1}{2(1)} \int_0^2 s ds = \frac{1}{2} [\frac{s^2}{2}]_0^2 = \frac{1}{4}[2^2-0^2] = \frac{4}{4} = 1$.

$u(1,1)=1$.

Question 15:

Solve $u_{xx}+u_{yy}=0$ for $x \in (0, \pi), y \in (0, \infty)$, $u(0,y)=0, u(\pi,y)=0$, $u(x,0)=f(x)$, and assume $u$ is bounded as $y\to\infty$.

Solution:

Separation of variables. $X''+\lambda X=0, X(0)=X(\pi)=0 \implies \lambda_n=n^2, X_n=\sin(nx)$.

$Y''-\lambda_n Y=0 \implies Y''-n^2 Y = 0$.

General solution $Y_n(y) = C_n e^{ny} + D_n e^{-ny}$.

For $u$ to be bounded as $y\to\infty$, the $e^{ny}$ term (which grows unboundedly) must be zero. So $C_n=0$.

$Y_n(y) = D_n e^{-ny}$.

General solution: $u(x,y) = \sum_{n=1}^\infty D_n e^{-ny} \sin(nx)$.

Apply $u(x,0)=f(x)$: $f(x) = \sum D_n e^0 \sin(nx) = \sum D_n \sin(nx)$.

$D_n = \frac{2}{\pi} \int_0^\pi f(x)\sin(nx) dx$.

$u(x,y) = \sum_{n=1}^\infty \left[ \frac{2}{\pi} \int_0^\pi f(s)\sin(ns) ds \right] e^{-ny} \sin(nx)$.

Question 16:

Solve the IVP $u_t + u_x = 0$, $u(x,0) = 1/(1+x^2)$.

Solution:

Characteristics: $\frac{dt}{1} = \frac{dx}{1} = \frac{du}{0}$.

$u=C_1$. $x-t=C_2$. General solution $u(x,t) = F(x-t)$.

$u(x,0) = F(x-0) = F(x) = 1/(1+x^2)$.

Solution $u(x,t) = \frac{1}{1+(x-t)^2}$.

Question 17:

Classify $u_{xx} + 2u_{xy} + \sin^2(x) u_{yy} = 0$.

Solution:

$A=1, B=2, C=\sin^2(x)$.

$\Delta = B^2 - 4AC = 2^2 - 4(1)(\sin^2 x) = 4 - 4\sin^2 x = 4(1-\sin^2 x) = 4\cos^2 x$.

  • If $\cos x \neq 0$ (i.e., $x \neq \frac{\pi}{2} + k\pi$), $\Delta > 0 \implies$ Hyperbolic.
  • If $\cos x = 0$ (i.e., $x = \frac{\pi}{2} + k\pi$), $\Delta = 0 \implies$ Parabolic.

Question 18:

Find the solution to $u_{tt} = 16u_{xx}$ on $x \in (0, \pi)$, $u(0,t)=u(\pi,t)=0$, $u(x,0)=\sin(3x)$, $u_t(x,0)=8\sin(x)$.

Solution:

$c=4, L=\pi$. $\omega_n=nc=4n$.

$f(x)=\sin(3x) \implies A_3=1, A_n=0$ for $n\neq 3$.

$g(x)=8\sin(x) \implies B_1 \omega_1 = 8, B_n=0$ for $n\neq 1$.

$\omega_1 = 4(1)=4$. $B_1(4)=8 \implies B_1=2$.

$u(x,t) = [A_1\cos(\omega_1 t)+B_1\sin(\omega_1 t)]\sin(x) + [A_3\cos(\omega_3 t)+B_3\sin(\omega_3 t)]\sin(3x) + \dots$

$u(x,t) = [0\cos(4t)+2\sin(4t)]\sin(x) + [1\cos(12t)+0\sin(12t)]\sin(3x)$.

$u(x,t) = 2\sin(4t)\sin(x) + \cos(12t)\sin(3x)$.

Question 19:

Find the Fourier Sine coefficient $B_2$ for $f(x)=x$ on $[0, \pi]$.

Solution:

$L=\pi$. $B_n = \frac{2(-1)^{n+1}}{n}$. (From Day 6, Q20, adjusting for L).

For $n=2$, $B_2 = \frac{2(-1)^{2+1}}{2} = \frac{2(-1)}{2} = -1$.

Question 20:

Solve $u_{xx}+u_{yy}=0$ for $x \in (0, 1), y \in (0, 2)$, $u(x,0)=0, u(x,2)=0, u(0,y)=y(2-y), u(1,y)=0$.

Solution:

$a=1, b=2$. Homogeneous on $x=1, y=0, y=2$. Non-homogeneous on $x=0$. $g(y)=y(2-y)$.

Separate: $Y''+\lambda Y=0, Y(0)=Y(2)=0 \implies \lambda_n=(n\pi/2)^2, Y_n=\sin(n\pi y/2)$.

$X'' - (n\pi/2)^2 X = 0$. $X_n(x) = C_n \cosh(n\pi x/2) + D_n \sinh(n\pi x/2)$.

Apply $u(1,y)=0 \implies X(1)=0$. Leads to $X_n(x)=E_n \sinh(n\pi(1-x)/2)$.

$u(x,y) = \sum E_n \sinh(n\pi(1-x)/2)\sin(n\pi y/2)$.

$u(0,y) = \sum E_n \sinh(n\pi/2)\sin(n\pi y/2) = y(2-y)$.

$E_n \sinh(n\pi/2)$ are Fourier Sine coefficients of $y(2-y)$ on $[0, 2]$.

$E_n \sinh(n\pi/2) = \frac{2}{2} \int_0^2 (2y-y^2)\sin(n\pi y/2) dy$.

Use integration by parts twice. $\int (2y-y^2)\sin(\dots) dy = [-\frac{2(2y-y^2)}{n\pi}\cos(\dots)] + \int \frac{2(2-2y)}{n\pi}\cos(\dots) dy$.

$= \dots + [\frac{4(1-y)}{(n\pi)^2}\sin(\dots)] - \int \frac{4(1-y)}{(n\pi)^2}\sin(\dots) (\frac{-2}{n\pi}) dy$? No.

$= [-\frac{2(2y-y^2)}{n\pi}\cos(\dots)] + [\frac{4(1-y)}{(n\pi)^2} \sin(\dots)] - \int \frac{-4}{(n\pi)^2}\sin(\dots) dy$.

$= [-\frac{2(2y-y^2)}{n\pi}\cos(\dots)] + [\frac{4(1-y)}{(n\pi)^2} \sin(\dots)] + [\frac{8}{(n\pi)^3}\cos(\dots)]_0^2$.

At $y=2$: $[0] + [0] + [\frac{8}{(n\pi)^3}\cos(n\pi)]$. At $y=0$: $[0] + [0] + [\frac{8}{(n\pi)^3}\cos(0)]$.

Integral value $= \frac{8}{(n\pi)^3}((-1)^n - 1)$.

$E_n \sinh(n\pi/2) = \frac{8((-1)^n - 1)}{(n\pi)^3}$.

$E_n = 0$ for even $n$. For odd $n=2m-1$, $E_{2m-1} \sinh((2m-1)\pi/2) = \frac{-16}{((2m-1)\pi)^3}$.

$E_{2m-1} = \frac{-16}{((2m-1)\pi)^3 \sinh((2m-1)\pi/2)}$.

$u(x,y) = \sum_{m=1}^\infty \frac{-16}{((2m-1)\pi)^3 \sinh(\frac{(2m-1)\pi}{2})} \sinh(\frac{(2m-1)\pi(1-x)}{2}) \sin(\frac{(2m-1)\pi y}{2})$.

Question 21:

Classify $u_{xx} - 6u_{xy} + 10u_{yy} = 0$.

Solution:

$A=1, B=-6, C=10$.

$\Delta = (-6)^2 - 4(1)(10) = 36 - 40 = -4$.

Elliptic.

Question 22:

Solve $u_t = 5 u_{xx}$, $u(0,t)=u(\pi,t)=0$, $u(x,0)=10$.

Solution:

$k=5, L=\pi$. $\lambda_n=n^2, X_n=\sin(nx)$.

$u(x,t) = \sum B_n e^{-5n^2 t}\sin(nx)$.

$u(x,0)=\sum B_n \sin(nx) = 10$.

$B_n$ are Fourier Sine coeff. of 10 on $[0,\pi]$. $B_n = 10 \times (\frac{2(1-(-1)^n)}{n\pi})$.

$B_n=0$ for even $n$. $B_{2m-1} = 10 \times \frac{4}{(2m-1)\pi} = \frac{40}{(2m-1)\pi}$.

$u(x,t) = \sum_{m=1}^\infty \frac{40}{(2m-1)\pi} e^{-5(2m-1)^2 t} \sin((2m-1)x)$.

Question 23:

Find the general solution to $u_t + u_x = t$.

Solution:

Characteristics: $\frac{dt}{1} = \frac{dx}{1} = \frac{du}{t}$.

$x-t=C_1$.

From $\frac{dt}{1} = \frac{du}{t}$, $du=tdt$. Integrating gives $u = \frac{t^2}{2} + C_2$. So $u - t^2/2 = C_2$.

General solution $u - t^2/2 = F(x-t)$, so $u(x,t) = \frac{t^2}{2} + F(x-t)$.

Question 24:

Find $u(0,1)$ for $u_{tt} = 4u_{xx}$, $u(x,0)=\cos(\pi x)$, $u_t(x,0)=0$.

Solution:

D'Alembert. $c=2, f(x)=\cos(\pi x), g(x)=0$. Point $(0,1)$.

$x+ct = 0+2(1)=2$. $x-ct = 0-2(1)=-2$.

$u(0,1) = \frac{1}{2}[f(2)+f(-2)] + 0 = \frac{1}{2}[\cos(2\pi)+\cos(-2\pi)] = \frac{1}{2}[1+1] = 1$.

Question 25:

Solve $u_{xx}+u_{yy}=0$ for $x\in(0,\pi), y\in(0,\pi)$, $u(x,0)=\sin x, u(x,\pi)=0, u(0,y)=0, u(\pi,y)=0$.

Solution:

$a=\pi, b=\pi$. Homogeneous on $x=0, x=\pi, y=\pi$. Non-homogeneous on $y=0$. $f(x)=\sin x$.

Use setup from Q10.

$X_n=\sin(nx)$. $Y_n(y) = E_n \sinh(n(\pi-y))$.

$u(x,y) = \sum E_n \sinh(n(\pi-y))\sin(nx)$.

$u(x,0) = \sum E_n \sinh(n\pi)\sin(nx) = \sin(x)$.

Matching: $n=1$. $E_1 \sinh(\pi)=1$. $E_n=0$ for $n\neq 1$.

$E_1 = 1/\sinh(\pi)$.

$u(x,y) = \frac{\sinh(\pi-y)\sin x}{\sinh \pi}$.

Question 26:

Classify $u_{tt} + 4u_{xx} = 0$.

Solution:

Let $y \to t$. $A=4, B=0, C=1$.

$\Delta = B^2 - 4AC = 0^2 - 4(4)(1) = -16$.

Since $\Delta < 0$, the PDE is Elliptic.

Question 27:

Solve $u_t = u_{xx}$, $u(0,t)=u(1,t)=0$, $u(x,0)=f(x)$. What is the long-term behavior ($t\to\infty$)?

Solution:

The solution is $u(x,t) = \sum B_n e^{-(n\pi)^2 t} \sin(n\pi x)$.

As $t\to\infty$, the exponential term $e^{-(n\pi)^2 t} \to 0$ for all $n \ge 1$.

So, $\lim_{t\to\infty} u(x,t) = 0$.

Question 28:

Solve $u_{tt}=u_{xx}$, $u(0,t)=u(\pi,t)=0$, $u(x,0)=\sin(5x)$, $u_t(x,0)=0$.

Solution:

$c=1, L=\pi$. $\omega_n=n$. $f(x)=\sin(5x) \implies A_5=1, A_n=0$ else. $g(x)=0 \implies B_n=0$ all $n$.

$u(x,t) = [A_5\cos(\omega_5 t)+B_5\sin(\omega_5 t)]\sin(5x)$.

$u(x,t) = [1\cos(5t)+0]\sin(5x) = \cos(5t)\sin(5x)$.

Question 29:

Solve $u_{xx}+u_{yy}=0$ for $x\in(0,a), y\in(0,b)$, $u(x,0)=0, u(x,b)=1, u(0,y)=0, u(a,y)=0$.

Solution:

Homogeneous on $x=0, x=a, y=0$. Non-homogeneous on $y=b$. $f(x)=1$.

Use theory setup.

$X_n=\sin(n\pi x/a)$. $Y_n(y)=E_n \sinh(n\pi y/a)$.

$u(x,y) = \sum E_n \sinh(n\pi y/a)\sin(n\pi x/a)$.

$u(x,b) = \sum E_n \sinh(n\pi b/a)\sin(n\pi x/a) = 1$.

$E_n \sinh(n\pi b/a)$ are Fourier Sine coeff. of $f(x)=1$ on $[0,a]$.

$E_n \sinh(n\pi b/a) = \frac{2(1-(-1)^n)}{n\pi}$.

$E_n=0$ for even $n$. $E_{2m-1} = \frac{4}{(2m-1)\pi \sinh((2m-1)\pi b/a)}$.

$u(x,y) = \sum_{m=1}^\infty \frac{4}{(2m-1)\pi \sinh(\frac{(2m-1)\pi b}{a})} \sinh(\frac{(2m-1)\pi y}{a}) \sin(\frac{(2m-1)\pi x}{a})$.

Wave Equation: Method of Extensions (Semi-Infinite Domain)

Theory Focus

We consider the 1D Wave Equation on a semi-infinite domain $x \in (0, \infty), t \in (0, \infty)$:

$$ u_{tt} = c^2 u_{xx} $$

with initial conditions:

$$ u(x,0) = f(x), \quad u_t(x,0) = g(x) \quad \text{for } x \in (0, \infty) $$

and a boundary condition at $x=0$. Common types are:

  • Dirichlet BC (Fixed End): $u(0,t) = 0$
  • Neumann BC (Free End): $u_x(0,t) = 0$

Method of Extensions

The idea is to extend the initial conditions $f(x)$ and $g(x)$ (originally defined only for $x \in (0, \infty)$) to the entire real line $\mathbb{R}$ in such a way that the solution to the wave equation on the infinite line, using these extended conditions with D'Alembert's formula, automatically satisfies the boundary condition at $x=0$.

Case 1: Dirichlet Boundary Condition $u(0,t) = 0$

We use **odd extensions** for the initial conditions:

$$ f_{odd}(x) = \begin{cases} f(x) & \text{if } x \in (0, \infty) \\ 0 & \text{if } x = 0 \\ -f(-x) & \text{if } x \in (-\infty, 0) \end{cases} $$ $$ g_{odd}(x) = \begin{cases} g(x) & \text{if } x \in (0, \infty) \\ 0 & \text{if } x = 0 \\ -g(-x) & \text{if } x \in (-\infty, 0) \end{cases} $$

The odd extension makes the function zero at the origin (if $f, g$ are continuous towards 0) and antisymmetric about the origin.

Case 2: Neumann Boundary Condition $u_x(0,t) = 0$

We use **even extensions** for the initial conditions:

$$ f_{even}(x) = \begin{cases} f(x) & \text{if } x \in [0, \infty) \\ f(-x) & \text{if } x \in (-\infty, 0) \end{cases} $$ $$ g_{even}(x) = \begin{cases} g(x) & \text{if } x \in [0, \infty) \\ g(-x) & \text{if } x \in (-\infty, 0) \end{cases} $$

The even extension makes the function symmetric about the origin, ensuring its derivative is zero at the origin (if $f', g'$ exist and are continuous towards 0).

Solution using D'Alembert's Formula

Once the appropriate extensions $f_{ext}$ and $g_{ext}$ (either odd or even) are defined, the solution for the semi-infinite problem for $x \in (0, \infty)$ is given by applying D'Alembert's formula to these extended functions:

$$ u(x,t) = \frac{1}{2}[f_{ext}(x+ct) + f_{ext}(x-ct)] + \frac{1}{2c} \int_{x-ct}^{x+ct} g_{ext}(s) \, ds \quad \text{for } x \in (0, \infty), t \in (0, \infty) $$

When evaluating $f_{ext}$ or $g_{ext}$ at negative arguments (which happens when $x-ct \in (-\infty, 0)$), use the appropriate odd or even extension definition.

For example, if $x-ct \in (-\infty, 0)$ and we used an odd extension for $f$: $f_{ext}(x-ct) = -f(-(x-ct)) = -f(ct-x)$.

If $x-ct \in (-\infty, 0)$ and we used an even extension for $f$: $f_{ext}(x-ct) = f(-(x-ct)) = f(ct-x)$.

Practice Questions

Question 1:

Let $f(x) = x^2$ for $x \in (0, \infty)$. Find its odd extension $f_{odd}(x)$ for all $x \in \mathbb{R}$.

Solution:

For $x \in (0, \infty)$, $f_{odd}(x) = f(x) = x^2$.

For $x \in (-\infty, 0)$, $f_{odd}(x) = -f(-x) = -((-x)^2) = -x^2$.

For $x=0$, $f_{odd}(0) = 0$.

$$ f_{odd}(x) = \begin{cases} x^2 & \text{if } x \in (0, \infty) \\ 0 & \text{if } x = 0 \\ -x^2 & \text{if } x \in (-\infty, 0) \end{cases} $$

This can also be written as $f_{odd}(x) = x|x|$ or $f_{odd}(x) = x^2 \text{sgn}(x)$.

Question 2:

Let $f(x) = x^2$ for $x \in (0, \infty)$. Find its even extension $f_{even}(x)$ for all $x \in \mathbb{R}$.

Solution:

For $x \in [0, \infty)$, $f_{even}(x) = f(x) = x^2$.

For $x \in (-\infty, 0)$, $f_{even}(x) = f(-x) = (-x)^2 = x^2$.

So, $f_{even}(x) = x^2$ for all $x \in \mathbb{R}$.

Question 3:

Let $g(x) = \sin(x)$ for $x \in (0, \infty)$. Find its odd extension $g_{odd}(x)$.

Solution:

For $x \in (0, \infty)$, $g_{odd}(x) = g(x) = \sin(x)$.

For $x \in (-\infty, 0)$, $g_{odd}(x) = -g(-x) = -\sin(-x) = -(-\sin x) = \sin x$.

For $x=0$, $g_{odd}(0) = 0$.

So, $g_{odd}(x) = \sin(x)$ for all $x \in \mathbb{R}$. (Sine is naturally an odd function).

Question 4:

Let $g(x) = \cos(x)$ for $x \in (0, \infty)$. Find its even extension $g_{even}(x)$.

Solution:

For $x \in [0, \infty)$, $g_{even}(x) = g(x) = \cos(x)$.

For $x \in (-\infty, 0)$, $g_{even}(x) = g(-x) = \cos(-x) = \cos x$.

So, $g_{even}(x) = \cos(x)$ for all $x \in \mathbb{R}$. (Cosine is naturally an even function).

Question 5:

Solve $u_{tt} = 9u_{xx}$ for $x \in (0, \infty), t \in (0, \infty)$, with $u(0,t)=0$, $u(x,0)=\sin(x)$, $u_t(x,0)=0$.

Solution:

$c=3$. Dirichlet BC $u(0,t)=0 \implies$ use odd extensions.

$f(x)=\sin x$ for $x \in (0, \infty)$. $f_{odd}(x)=\sin x$ for all $x$.

$g(x)=0$ for $x \in (0, \infty)$. $g_{odd}(x)=0$ for all $x$.

Use D'Alembert with extended functions:

$$ u(x,t) = \frac{1}{2}[f_{odd}(x+3t) + f_{odd}(x-3t)] + \frac{1}{2(3)} \int_{x-3t}^{x+3t} 0 \, ds $$ $$ u(x,t) = \frac{1}{2}[\sin(x+3t) + \sin(x-3t)] $$

Using $\sin A + \sin B = 2\sin(\frac{A+B}{2})\cos(\frac{A-B}{2})$:

$$ u(x,t) = \frac{1}{2}[2\sin(\frac{x+3t+x-3t}{2})\cos(\frac{x+3t-(x-3t)}{2})] $$ $$ u(x,t) = \sin(x)\cos(3t) $$

Solution is $u(x,t) = \sin(x)\cos(3t)$ for $x \in (0, \infty), t \in (0, \infty)$.

Question 6:

Solve $u_{tt} = 9u_{xx}$ for $x \in (0, \infty), t \in (0, \infty)$, with $u_x(0,t)=0$, $u(x,0)=\cos(x)$, $u_t(x,0)=0$.

Solution:

$c=3$. Neumann BC $u_x(0,t)=0 \implies$ use even extensions.

$f(x)=\cos x$ for $x \in (0, \infty)$. $f_{even}(x)=\cos x$ for all $x$.

$g(x)=0$ for $x \in (0, \infty)$. $g_{even}(x)=0$ for all $x$.

Use D'Alembert with extended functions:

$$ u(x,t) = \frac{1}{2}[f_{even}(x+3t) + f_{even}(x-3t)] + 0 $$ $$ u(x,t) = \frac{1}{2}[\cos(x+3t) + \cos(x-3t)] $$

Using $\cos A + \cos B = 2\cos(\frac{A+B}{2})\cos(\frac{A-B}{2})$:

$$ u(x,t) = \frac{1}{2}[2\cos(\frac{x+3t+x-3t}{2})\cos(\frac{x+3t-(x-3t)}{2})] $$ $$ u(x,t) = \cos(x)\cos(3t) $$

Solution is $u(x,t) = \cos(x)\cos(3t)$ for $x \in (0, \infty), t \in (0, \infty)$.

Question 7:

Let $f(x)=x$ for $x \in (0, \infty)$. Find $f_{odd}(x)$ and $f_{even}(x)$.

Solution:

Odd extension:

$f_{odd}(x) = x$ for $x \in (0, \infty)$. $f_{odd}(0)=0$. $f_{odd}(x)=-f(-x)=-(-x)=x$ for $x \in (-\infty, 0)$.

So, $f_{odd}(x)=x$ for all $x \in \mathbb{R}$.

Even extension:

$f_{even}(x) = x$ for $x \in [0, \infty)$. $f_{even}(x)=f(-x)=-x$ for $x \in (-\infty, 0)$.

So, $f_{even}(x)=|x|$ for all $x \in \mathbb{R}$.

Question 8:

Solve $u_{tt} = c^2 u_{xx}$ for $x \in (0, \infty), t \in (0, \infty)$, $u(0,t)=0$, $u(x,0)=x$, $u_t(x,0)=0$.

Solution:

Dirichlet BC $\implies$ use odd extensions.

$f(x)=x$ for $x \in (0, \infty)$. $f_{odd}(x)=x$ for all $x$.

$g(x)=0$ for $x \in (0, \infty)$. $g_{odd}(x)=0$ for all $x$.

$u(x,t) = \frac{1}{2}[f_{odd}(x+ct) + f_{odd}(x-ct)] + 0$.

$u(x,t) = \frac{1}{2}[(x+ct) + (x-ct)] = \frac{1}{2}[2x] = x$.

Solution is $u(x,t) = x$ for $x \in (0, \infty), t \in (0, \infty)$.

Question 9:

Solve $u_{tt} = c^2 u_{xx}$ for $x \in (0, \infty), t \in (0, \infty)$, $u_x(0,t)=0$, $u(x,0)=x$, $u_t(x,0)=0$.

Solution:

Neumann BC $\implies$ use even extensions.

$f(x)=x$ for $x \in (0, \infty)$. $f_{even}(x)=|x|$ for all $x$.

$g(x)=0$ for $x \in (0, \infty)$. $g_{even}(x)=0$ for all $x$.

$u(x,t) = \frac{1}{2}[f_{even}(x+ct) + f_{even}(x-ct)] + 0$.

$u(x,t) = \frac{1}{2}[|x+ct| + |x-ct|]$.

Since we are only interested in $x \in (0, \infty), t \in (0, \infty)$, $x+ct$ is always positive, so $|x+ct|=x+ct$.

The term $|x-ct|$ depends on whether $x \ge ct$ or $x \in (0, ct)$.

$u(x,t) = \frac{1}{2}(x+ct + |x-ct|)$.

If $x \ge ct$, $u(x,t) = \frac{1}{2}(x+ct + x-ct) = \frac{1}{2}(2x) = x$.

If $x \in (0, ct)$, $u(x,t) = \frac{1}{2}(x+ct + -(x-ct)) = \frac{1}{2}(x+ct -x+ct) = \frac{1}{2}(2ct) = ct$.

$$ u(x,t) = \begin{cases} x & \text{if } x \ge ct \\ ct & \text{if } x \in (0, ct) \end{cases} $$

Question 10:

Solve $u_{tt} = 4u_{xx}$ for $x \in (0, \infty), t \in (0, \infty)$, $u(0,t)=0$, $u(x,0)=0$, $u_t(x,0)=x$.

Solution:

$c=2$. Dirichlet BC $\implies$ odd extensions.

$f(x)=0 \implies f_{odd}(x)=0$.

$g(x)=x$ for $x \in (0, \infty)$. $g_{odd}(x)=x$ for all $x$.

$u(x,t) = 0 + \frac{1}{2c} \int_{x-ct}^{x+ct} g_{odd}(s) ds = \frac{1}{4} \int_{x-2t}^{x+2t} s ds$.

$u(x,t) = \frac{1}{4} [\frac{s^2}{2}]_{x-2t}^{x+2t} = \frac{1}{8} [(x+2t)^2 - (x-2t)^2]$.

$u(x,t) = \frac{1}{8} [(x^2+4xt+4t^2) - (x^2-4xt+4t^2)] = \frac{1}{8}[8xt] = xt$.

Solution is $u(x,t) = xt$ for $x \in (0, \infty), t \in (0, \infty)$.

Question 11:

Solve $u_{tt} = 4u_{xx}$ for $x \in (0, \infty), t \in (0, \infty)$, $u_x(0,t)=0$, $u(x,0)=0$, $u_t(x,0)=x$.

Solution:

$c=2$. Neumann BC $\implies$ even extensions.

$f(x)=0 \implies f_{even}(x)=0$.

$g(x)=x$ for $x \in (0, \infty)$. $g_{even}(x)=|x|$ for all $x$.

$u(x,t) = 0 + \frac{1}{2c} \int_{x-ct}^{x+ct} g_{even}(s) ds = \frac{1}{4} \int_{x-2t}^{x+2t} |s| ds$.

Case 1: $x-2t \ge 0$ (i.e., $x \ge 2t$). The interval $[x-2t, x+2t]$ is entirely non-negative. $u(x,t) = \frac{1}{4} \int_{x-2t}^{x+2t} s ds = \frac{1}{4} [\frac{s^2}{2}]_{x-2t}^{x+2t} = \frac{1}{8}[(x+2t)^2 - (x-2t)^2] = \frac{1}{8}[8xt] = xt$.

Case 2: $x-2t < 0$ (i.e., $x \in (0, 2t)$). The interval contains 0. $u(x,t) = \frac{1}{4} \left[ \int_{x-2t}^{0} |s| ds + \int_{0}^{x+2t} |s| ds \right] = \frac{1}{4} \left[ \int_{x-2t}^{0} (-s) ds + \int_{0}^{x+2t} s ds \right]$.

$u(x,t) = \frac{1}{4} \left[ [-\frac{s^2}{2}]_{x-2t}^{0} + [\frac{s^2}{2}]_{0}^{x+2t} \right]$.

$u(x,t) = \frac{1}{4} \left[ (0 - (-\frac{(x-2t)^2}{2})) + (\frac{(x+2t)^2}{2} - 0) \right]$.

$u(x,t) = \frac{1}{8} [(x-2t)^2 + (x+2t)^2] = \frac{1}{8} [(x^2-4xt+4t^2) + (x^2+4xt+4t^2)]$.

$u(x,t) = \frac{1}{8} [2x^2 + 8t^2] = \frac{x^2+4t^2}{4}$.

$$ u(x,t) = \begin{cases} xt & \text{if } x \ge 2t \\ \frac{x^2+4t^2}{4} & \text{if } x \in (0, 2t) \end{cases} $$

Question 12:

Let $f(x)=1$ for $x \in (0, \infty)$. Find $f_{odd}(x)$ and $f_{even}(x)$.

Solution:

Odd extension: $f_{odd}(x)=1$ for $x \in (0, \infty)$, $f_{odd}(0)=0$, $f_{odd}(x)=-f(-x)=-1$ for $x \in (-\infty, 0)$. This is the sign function, $f_{odd}(x)=\text{sgn}(x)$.

Even extension: $f_{even}(x)=1$ for $x \in [0, \infty)$, $f_{even}(x)=f(-x)=1$ for $x \in (-\infty, 0)$. So $f_{even}(x)=1$ for all $x \in \mathbb{R}$.

Question 13:

Solve $u_{tt}=u_{xx}$, for $x \in (0, \infty)$, $u(0,t)=0$, $u(x,0)=0$, $u_t(x,0)=1$.

Solution:

$c=1$. Dirichlet $\implies$ odd.

$f(x)=0 \implies f_{odd}=0$. $g(x)=1$ for $x \in (0, \infty)$. $g_{odd}(x)=\text{sgn}(x)$.

$u(x,t) = 0 + \frac{1}{2(1)} \int_{x-t}^{x+t} \text{sgn}(s) ds$.

Case 1: $x-t \ge 0$ ($x \ge t$). Interval is $\subseteq [0, \infty)$. $\int_{x-t}^{x+t} 1 ds = [s]_{x-t}^{x+t} = (x+t)-(x-t)=2t$. $u(x,t)=\frac{1}{2}(2t)=t$.

Case 2: $x-t < 0$ ($x \in (0, t)$). Interval is $(-,+)$. $\int_{x-t}^{x+t} \text{sgn}(s) ds = \int_{x-t}^{0} (-1) ds + \int_{0}^{x+t} (1) ds$.

$= [-s]_{x-t}^0 + [s]_0^{x+t} = (0 - (-(x-t))) + ((x+t)-0) = (x-t) + (x+t) = 2x$.

$u(x,t)=\frac{1}{2}(2x)=x$.

$$ u(x,t) = \begin{cases} t & \text{if } x \ge t \\ x & \text{if } x \in (0, t) \end{cases} $$

This can be written as $u(x,t) = \min(x,t)$.

Question 14:

Solve $u_{tt}=u_{xx}$, for $x \in (0, \infty)$, $u_x(0,t)=0$, $u(x,0)=0$, $u_t(x,0)=1$.

Solution:

$c=1$. Neumann $\implies$ even.

$f(x)=0 \implies f_{even}=0$. $g(x)=1$ for $x \in (0, \infty)$. $g_{even}(x)=1$ for all $x$.

$u(x,t) = 0 + \frac{1}{2(1)} \int_{x-t}^{x+t} 1 ds$.

$u(x,t) = \frac{1}{2} [s]_{x-t}^{x+t} = \frac{1}{2}[(x+t)-(x-t)] = \frac{1}{2}(2t) = t$.

Solution is $u(x,t)=t$ for $x \in (0, \infty), t \in (0, \infty)$.

Question 15:

Let $f(x) = e^{-x}$ for $x \in (0, \infty)$. Find $f_{odd}(x)$ and $f_{even}(x)$.

Solution:

Odd: $f_{odd}(x)=e^{-x}$ for $x \in (0, \infty)$, $f_{odd}(0)=0$, $f_{odd}(x)=-f(-x)=-e^{-(-x)}=-e^x$ for $x \in (-\infty, 0)$. $$ f_{odd}(x) = \begin{cases} e^{-x} & x \in (0, \infty) \\ 0 & x=0 \\ -e^x & x \in (-\infty, 0) \end{cases} $$

Even: $f_{even}(x)=e^{-x}$ for $x \in [0, \infty)$, $f_{even}(x)=f(-x)=e^{-(-x)}=e^x$ for $x \in (-\infty, 0)$. $$ f_{even}(x) = \begin{cases} e^{-x} & x \in [0, \infty) \\ e^x & x \in (-\infty, 0) \end{cases} $$

This is $f_{even}(x)=e^{-|x|}$.

Question 16:

Solve $u_{tt}=u_{xx}$, for $x \in (0, \infty)$, $u(0,t)=0$, $u(x,0)=e^{-x}$, $u_t(x,0)=0$.

Solution:

$c=1$. Dirichlet $\implies$ odd.

$f_{odd}(x)$ from Q15. $g_{odd}(x)=0$.

$u(x,t) = \frac{1}{2}[f_{odd}(x+t) + f_{odd}(x-t)]$.

Case 1: $x-t \ge 0$ ($x \ge t$). Both arguments are non-negative. $u(x,t) = \frac{1}{2}[e^{-(x+t)} + e^{-(x-t)}] = \frac{1}{2}e^{-x}[e^{-t}+e^t] = e^{-x}\cosh(t)$.

Case 2: $x-t < 0$ ($x \in (0, t)$). $x+t>0$, $x-t<0$. $u(x,t) = \frac{1}{2}[f_{odd}(x+t) + f_{odd}(x-t)] = \frac{1}{2}[e^{-(x+t)} + (-e^{x-t})]$.

$u(x,t) = \frac{1}{2}[e^{-x}e^{-t} - e^x e^{-t}] = \frac{1}{2}e^{-t}[e^{-x}-e^x] = e^{-t} \frac{e^{-x}-e^x}{2} = -e^{-t}\sinh(x)$.

$$ u(x,t) = \begin{cases} e^{-x}\cosh(t) & \text{if } x \ge t \\ -e^{-t}\sinh(x) & \text{if } x \in (0, t) \end{cases} $$

Question 17:

Solve $u_{tt}=u_{xx}$, for $x \in (0, \infty)$, $u_x(0,t)=0$, $u(x,0)=e^{-x}$, $u_t(x,0)=0$.

Solution:

$c=1$. Neumann $\implies$ even.

$f_{even}(x)=e^{-|x|}$. $g_{even}(x)=0$.

$u(x,t) = \frac{1}{2}[f_{even}(x+t) + f_{even}(x-t)] = \frac{1}{2}[e^{-|x+t|} + e^{-|x-t|}]$.

Since $x>0, t>0$, $x+t>0$, so $|x+t|=x+t$.

$$ u(x,t) = \frac{1}{2}[e^{-(x+t)} + e^{-|x-t|}] $$

Case 1: $x \ge t$. $|x-t|=x-t$. $u(x,t) = \frac{1}{2}[e^{-(x+t)} + e^{-(x-t)}] = e^{-x}\cosh(t)$.

Case 2: $x \in (0,t)$. $|x-t|=-(x-t)=t-x$. $u(x,t) = \frac{1}{2}[e^{-(x+t)} + e^{-(t-x)}] = \frac{1}{2}e^{-t}[e^{-x} + e^{x}] = e^{-t}\cosh(x)$.

$$ u(x,t) = \begin{cases} e^{-x}\cosh(t) & \text{if } x \ge t \\ e^{-t}\cosh(x) & \text{if } x \in (0, t) \end{cases} $$

Question 18:

Let $f(x)=H(x-a)$ for $x \in (0, \infty)$, where $a>0$ and $H$ is the Heaviside step function ($H(z)=1$ if $z>0$, 0 if $z \le 0$). Find $f_{odd}(x)$.

Solution:

$f(x)=1$ if $x \in (a, \infty)$, $f(x)=0$ if $x \in (0, a]$.

$f_{odd}(x)=f(x)$ for $x \in (0, \infty)$. $f_{odd}(0)=0$. $f_{odd}(x)=-f(-x)$ for $x \in (-\infty, 0)$.

If $x \in (-\infty, 0)$, then $-x \in (0, \infty)$.

If $-x > a$ (i.e., $x < -a$), $f(-x)=1$, so $f_{odd}(x)=-1$.

If $0 < -x \le a$ (i.e., $x \in [-a, 0)$), $f(-x)=0$, so $f_{odd}(x)=0$.

$$ f_{odd}(x) = \begin{cases} 1 & \text{if } x \in (a, \infty) \\ 0 & \text{if } x \in [-a, a] \\ -1 & \text{if } x \in (-\infty, -a) \end{cases} $$

Question 19:

Let $f(x)=H(x-a)$ for $x \in (0, \infty)$, where $a>0$. Find $f_{even}(x)$.

Solution:

$f(x)=1$ if $x \in (a, \infty)$, $f(x)=0$ if $x \in [0, a]$.

$f_{even}(x)=f(x)$ for $x \in [0, \infty)$. $f_{even}(x)=f(-x)$ for $x \in (-\infty, 0)$.

If $x \in (-\infty, 0)$, then $-x \in (0, \infty)$.

If $-x > a$ (i.e., $x < -a$), then $f(-x)=1$. So $f_{even}(x)=1$.

If $0 < -x \le a$ (i.e., $x \in [-a, 0)$), then $f(-x)=0$. So $f_{even}(x)=0$.

$$ f_{even}(x) = \begin{cases} 1 & \text{if } x \in (a, \infty) \\ 0 & \text{if } x \in [0, a] \\ 0 & \text{if } x \in [-a, 0) \\ 1 & \text{if } x \in (-\infty, -a) \end{cases} $$

This simplifies to:

$$ f_{even}(x) = \begin{cases} 1 & \text{if } |x|>a \\ 0 & \text{if } |x| \le a \end{cases} $$

Question 20:

Solve $u_{tt}=c^2 u_{xx}$, for $x \in (0, \infty)$, $u(0,t)=0$, $u(x,0)=f(x)$ from Q18, $u_t(x,0)=0$. Find $u(a/2, a/c)$.

Solution:

Dirichlet $\implies$ odd. $f_{odd}(x)$ from Q18. $g_{odd}=0$.

$u(x,t) = \frac{1}{2}[f_{odd}(x+ct) + f_{odd}(x-ct)]$.

We need $u(a/2, a/c)$. $x=a/2, t=a/c$.

$x+ct = a/2 + c(a/c) = a/2+a = 3a/2$.

$x-ct = a/2 - c(a/c) = a/2-a = -a/2$.

$u(a/2, a/c) = \frac{1}{2}[f_{odd}(3a/2) + f_{odd}(-a/2)]$.

From Q18: $f_{odd}(3a/2)=1$ (since $3a/2 > a$). $f_{odd}(-a/2)=0$ (since $x=-a/2 \in [-a, a]$).

$u(a/2, a/c) = \frac{1}{2}[1 + 0] = 1/2$.

Question 21:

Solve $u_{tt}=c^2 u_{xx}$, for $x \in (0, \infty)$, $u_x(0,t)=0$, $u(x,0)=f(x)$ from Q19, $u_t(x,0)=0$. Find $u(a/2, a/c)$.

Solution:

Neumann $\implies$ even. $f_{even}(x)$ from Q19. $g_{even}=0$.

$u(x,t) = \frac{1}{2}[f_{even}(x+ct) + f_{even}(x-ct)]$.

We need $u(a/2, a/c)$. $x=a/2, t=a/c$.

$x+ct = 3a/2$. $x-ct = -a/2$.

$u(a/2, a/c) = \frac{1}{2}[f_{even}(3a/2) + f_{even}(-a/2)]$.

From Q19: $f_{even}(3a/2)=1$ (since $|3a/2|>a$). $f_{even}(-a/2)=0$ (since $|-a/2|\le a$).

$u(a/2, a/c) = \frac{1}{2}[1 + 0] = 1/2$.

Question 22:

Solve $u_{tt}=u_{xx}$, for $x \in (0, \infty)$, $u(0,t)=0$, $u(x,0)=0$, $u_t(x,0)=\sin(\pi x)$.

Solution:

$c=1$. Dirichlet $\implies$ odd.

$f_{odd}=0$. $g(x)=\sin(\pi x)$ for $x \in (0, \infty)$. $g_{odd}(x)=\sin(\pi x)$ for all $x$.

$u(x,t) = 0 + \frac{1}{2(1)} \int_{x-t}^{x+t} \sin(\pi s) ds = \frac{1}{2} [-\frac{\cos(\pi s)}{\pi}]_{x-t}^{x+t}$.

$u(x,t) = -\frac{1}{2\pi} [\cos(\pi(x+t)) - \cos(\pi(x-t))]$.

Using $\cos A - \cos B = -2\sin(\frac{A+B}{2})\sin(\frac{A-B}{2})$:

$u(x,t) = -\frac{1}{2\pi} [-2 \sin(\pi x) \sin(\pi t)] = \frac{1}{\pi}\sin(\pi x)\sin(\pi t)$.

Question 23:

Question 24:

Let $f(x)=H(x-a)$ for $x \in (0, \infty)$, where $a>0$ and $H$ is the Heaviside step function ($H(z)=1$ if $z \in (0, \infty)$, 0 if $z \in (-\infty, 0]$). Find $f_{odd}(x)$.

Solution:

$f(x)=1$ if $x \in (a, \infty)$, $f(x)=0$ if $x \in (0, a]$.

$f_{odd}(x)=f(x)$ for $x \in (0, \infty)$. $f_{odd}(0)=0$. $f_{odd}(x)=-f(-x)$ for $x \in (-\infty, 0)$.

If $x \in (-\infty, 0)$, then $-x \in (0, \infty)$.

If $-x > a$ (i.e., $x < -a$), $f(-x)=1$, so $f_{odd}(x)=-1$.

If $0 < -x \le a$ (i.e., $x \in [-a, 0)$), $f(-x)=0$, so $f_{odd}(x)=0$.

$$ f_{odd}(x) = \begin{cases} 1 & \text{if } x \in (a, \infty) \\ 0 & \text{if } x \in [-a, a] \\ -1 & \text{if } x \in (-\infty, -a) \end{cases} $$

This can be written as $f_{odd}(x) = H(x-a) - H(-x-a)$.

Question 25:

Find $u(a/2, a/c)$ for the problem in Q20 using $f_{odd}$ from Q24.

Solution:

Problem is $u_{tt}=c^2 u_{xx}$, $x \in (0, \infty)$, $u(0,t)=0$, $u(x,0)=f(x)$ from Q24, $u_t(x,0)=0$. Find $u(a/2, a/c)$.

Dirichlet $\implies$ odd. $f_{odd}(x)$ from Q24. $g_{odd}=0$.

$u(x,t) = \frac{1}{2}[f_{odd}(x+ct) + f_{odd}(x-ct)]$.

$x=a/2, t=a/c$. $x+ct = a/2 + c(a/c) = a/2+a = 3a/2$. $x-ct = a/2 - c(a/c) = a/2-a = -a/2$.

$u(a/2, a/c) = \frac{1}{2}[f_{odd}(3a/2) + f_{odd}(-a/2)]$.

From Q24: $f_{odd}(3a/2) = 1$ (since $3a/2 > a$). $f_{odd}(-a/2) = 0$ (since $x=-a/2 \in [-a, a]$).

$u(a/2, a/c) = \frac{1}{2}[1 + 0] = 1/2$.

Question 26:

Why does the odd extension satisfy $u(0,t)=0$?

Solution:

The solution using D'Alembert is $u(x,t) = \frac{1}{2}[f_{odd}(x+ct) + f_{odd}(x-ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct} g_{odd}(s) ds$.

Set $x=0$: $u(0,t) = \frac{1}{2}[f_{odd}(ct) + f_{odd}(-ct)] + \frac{1}{2c}\int_{-ct}^{ct} g_{odd}(s) ds$.

Since $f_{odd}$ is odd, $f_{odd}(-ct) = -f_{odd}(ct)$. So $f_{odd}(ct) + f_{odd}(-ct) = f_{odd}(ct) - f_{odd}(ct) = 0$.

Since $g_{odd}$ is odd, its integral over a symmetric interval $[-ct, ct]$ is zero. $\int_{-A}^{A} \text{odd}(s) ds = 0$.

Therefore, $u(0,t) = \frac{1}{2}[0] + \frac{1}{2c}[0] = 0$.

Question 27:

Why does the even extension satisfy $u_x(0,t)=0$?

Solution:

$u(x,t) = \frac{1}{2}[f_{even}(x+ct) + f_{even}(x-ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct} g_{even}(s) ds$.

Differentiate with respect to $x$ using Chain Rule and Leibniz rule:

$u_x(x,t) = \frac{1}{2}[f'_{even}(x+ct) \cdot 1 + f'_{even}(x-ct) \cdot 1] + \frac{1}{2c}[g_{even}(x+ct)\cdot 1 - g_{even}(x-ct)\cdot 1]$.

Set $x=0$: $u_x(0,t) = \frac{1}{2}[f'_{even}(ct) + f'_{even}(-ct)] + \frac{1}{2c}[g_{even}(ct) - g_{even}(-ct)]$.

If $f_{even}$ is even, its derivative $f'_{even}$ is odd (assuming differentiability). Thus $f'_{even}(-ct) = -f'_{even}(ct)$. The first term is $\frac{1}{2}[f'_{even}(ct) - f'_{even}(ct)] = 0$.

Since $g_{even}$ is even, $g_{even}(-ct)=g_{even}(ct)$. The second term is $\frac{1}{2c}[g_{even}(ct) - g_{even}(ct)] = 0$.

Therefore, $u_x(0,t) = 0 + 0 = 0$.

Question 28:

$f(x) = \begin{cases} 2x-x^2 & x \in (0,2) \\ 0 & x \in [2, \infty) \end{cases}$. $g=0$.

$f_{odd}(x) = -f(-x)$ for $x<0$. If $x \in (-2, 0)$, then $-x \in (0, 2)$, so $f(-x) = 2(-x)-(-x)^2 = -2x-x^2$. $f_{odd}(x) = -(-2x-x^2) = 2x+x^2$. If $x \in (-\infty, -2]$, then $-x \in [2, \infty)$, so $f(-x)=0$. $f_{odd}(x)=0$.

$$ f_{odd}(x) = \begin{cases} 2x-x^2 & \text{if } x \in (0,2) \\ 2x+x^2 & \text{if } x \in (-2,0) \\ 0 & \text{otherwise } (|x| \ge 2 \text{ or } x=0) \end{cases} $$

$u(1,3) = \frac{1}{2}[0+0] = 0$.

Question 30:

If $f(x)=0$ and $g(x)=1$ for $x \in (0,L)$ and $g(x)=0$ for $x \in [L, \infty)$, find $u(L/2, L/c)$ for the wave equation on $x \in (0, \infty)$ with $u(0,t)=0$.

Solution:

Dirichlet $\implies$ odd.

$f_{odd}=0$. $g(x) = \begin{cases} 1 & x \in (0,L) \\ 0 & x \in [L, \infty) \end{cases}$.

$g_{odd}(x) = -g(-x)$ for $x<0$. If $x \in (-L, 0)$, then $-x \in (0, L)$, $g(-x)=1$, so $g_{odd}(x)=-1$. If $x \in (-\infty, -L]$, then $-x \in [L, \infty)$, $g(-x)=0$, so $g_{odd}(x)=0$.

$$ g_{odd}(x) = \begin{cases} 1 & \text{if } x \in (0,L) \\ -1 & \text{if } x \in (-L,0) \\ 0 & \text{otherwise } (|x| \ge L \text{ or } x=0) \end{cases} $$

$u(x,t) = \frac{1}{2c}\int_{x-ct}^{x+ct} g_{odd}(s) ds$.

Find $u(L/2, L/c)$. $x=L/2, t=L/c$.

$x+ct = L/2+c(L/c)=L/2+L=3L/2$. $x-ct = L/2-c(L/c)=L/2-L=-L/2$.

$u(L/2, L/c) = \frac{1}{2c} \int_{-L/2}^{3L/2} g_{odd}(s) ds$.

$ = \frac{1}{2c} \left[ \int_{-L/2}^{0} g_{odd}(s)ds + \int_{0}^{L} g_{odd}(s)ds + \int_{L}^{3L/2} g_{odd}(s)ds \right]$.

$ = \frac{1}{2c} \left[ \int_{-L/2}^{0} (-1) ds + \int_{0}^{L} (1) ds + \int_{L}^{3L/2} (0) ds \right]$.

$ = \frac{1}{2c} \left[ [-s]_{-L/2}^0 + [s]_0^L + 0 \right]$.

$ = \frac{1}{2c} \left[ (0 - (-(-L/2))) + (L-0) \right] = \frac{1}{2c} [ -L/2 + L ] = \frac{1}{2c} [L/2] = \frac{L}{4c}$.

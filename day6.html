Day 6: Separation of Variables (Laplace Eq. - Rectangle)

Theory Focus

Separation of Variables for Laplace's Equation ($u_{xx} + u_{yy} = 0$) in a Rectangle:

Solves the Boundary Value Problem (BVP) on a rectangular domain $x \in (0, a), y \in (0, b)$.

Strategy (Superposition): The general problem has boundary conditions on all four sides. The standard approach is to decompose the problem into four simpler problems, where only one side has a non-homogeneous boundary condition and the other three are homogeneous (zero). The solution to the original problem is the sum of the solutions to these four simpler problems.

Example: Solving for Non-Homogeneous Top Boundary

$$ u_{xx} + u_{yy} = 0 $$ BCs: $u(0,y) = 0$, $u(a,y) = 0$, $u(x,0) = 0$, $u(x,b) = f(x)$.

Steps:

  1. Assume Separable Solution: $u(x,y) = X(x)Y(y)$. Substitute into PDE: $X''Y + XY'' = 0$.
  2. Separate Variables: $\frac{X''}{X} = -\frac{Y''}{Y} = -\lambda$ (separation constant). $$ X'' + \lambda X = 0 $$ $$ Y'' - \lambda Y = 0 $$
  3. Apply Homogeneous Boundary Conditions to Spatial Problem (X): $u(0,y) = X(0)Y(y) = 0 \implies X(0) = 0$. $u(a,y) = X(a)Y(y) = 0 \implies X(a) = 0$. The eigenvalue problem for $X$ is $X'' + \lambda X = 0, X(0)=0, X(a)=0$. The solutions are $\lambda_n = (\frac{n\pi}{a})^2$, $X_n(x) = \sin(\frac{n\pi x}{a})$ for $n=1, 2, 3, \dots$.
  4. Solve the ODE for Y: For each $\lambda_n = (\frac{n\pi}{a})^2 > 0$, the ODE is $Y_n'' - \lambda_n Y_n = 0$. The general solution is: $$ Y_n(y) = C_n \cosh(\sqrt{\lambda_n} y) + D_n \sinh(\sqrt{\lambda_n} y) $$ $$ Y_n(y) = C_n \cosh\left(\frac{n\pi y}{a}\right) + D_n \sinh\left(\frac{n\pi y}{a}\right) $$
  5. Apply Remaining Homogeneous Boundary Condition to Y: $u(x,0) = X(x)Y(0) = 0 \implies Y(0)=0$. $Y_n(0) = C_n \cosh(0) + D_n \sinh(0) = C_n(1) + D_n(0) = C_n = 0$. So, $Y_n(y) = D_n \sinh(\frac{n\pi y}{a})$.
  6. Superposition Principle: The solution is a superposition of product solutions $u_n(x,y) = X_n(x)Y_n(y)$: $$ u(x,y) = \sum_{n=1}^{\infty} D_n \sinh\left(\frac{n\pi y}{a}\right) \sin\left(\frac{n\pi x}{a}\right) $$ (We absorb the constant $D_n$ into a single coefficient, let's call it $E_n$). $$ u(x,y) = \sum_{n=1}^{\infty} E_n \sinh\left(\frac{n\pi y}{a}\right) \sin\left(\frac{n\pi x}{a}\right) $$
  7. Apply Non-Homogeneous Boundary Condition using Fourier Sine Series: Set $y=b$: $$ u(x,b) = f(x) = \sum_{n=1}^{\infty} E_n \sinh\left(\frac{n\pi b}{a}\right) \sin\left(\frac{n\pi x}{a}\right) $$ This is a Fourier Sine Series for $f(x)$. The coefficients are $E_n \sinh\left(\frac{n\pi b}{a}\right)$. $$ E_n \sinh\left(\frac{n\pi b}{a}\right) = \frac{2}{a} \int_0^a f(x) \sin\left(\frac{n\pi x}{a}\right) \, dx $$ $$ E_n = \frac{1}{\sinh(n\pi b/a)} \left[ \frac{2}{a} \int_0^a f(x) \sin\left(\frac{n\pi x}{a}\right) \, dx \right] $$

Note: If the homogeneous BCs are on the $x$-variable sides, the signs in the ODEs flip, $X$ involves hyperbolic functions, and $Y$ involves sines/cosines.

Practice Questions

Question 1:

Find the general solution to $Y'' - 4Y = 0$.

Solution:

Characteristic equation: $r^2 - 4 = 0 \implies r = \pm 2$.

The general solution is $Y(y) = C_1 e^{2y} + C_2 e^{-2y}$.

Alternatively, it can be written using hyperbolic functions: $Y(y) = A \cosh(2y) + B \sinh(2y)$.

Question 2:

Find the general solution to $Y'' - \lambda Y = 0$ where $\lambda = (n\pi/a)^2 > 0$.

Solution:

Let $\mu = \sqrt{\lambda} = n\pi/a$. The ODE is $Y'' - \mu^2 Y = 0$.

Characteristic equation: $r^2 - \mu^2 = 0 \implies r = \pm \mu$.

General solution: $Y(y) = C_1 e^{\mu y} + C_2 e^{-\mu y} = C_1 e^{(n\pi/a)y} + C_2 e^{-(n\pi/a)y}$.

Alternative form: $Y(y) = A \cosh(\mu y) + B \sinh(\mu y) = A \cosh(\frac{n\pi y}{a}) + B \sinh(\frac{n\pi y}{a})$.

Question 3:

Solve $Y'' - 9Y = 0$ with $Y(0)=0$.

Solution:

General solution: $Y(y) = A \cosh(3y) + B \sinh(3y)$.

$Y(0) = A \cosh(0) + B \sinh(0) = A(1) + B(0) = A = 0$.

The solution is $Y(y) = B \sinh(3y)$.

Question 4:

Solve $X'' - (2\pi)^2 X = 0$ with $X(1)=0$. (Assume general solution $X(x) = A \cosh(2\pi x) + B \sinh(2\pi x)$).

Solution:

$X(1) = A \cosh(2\pi) + B \sinh(2\pi) = 0$.

This gives a relationship between A and B: $A = -B \frac{\sinh(2\pi)}{\cosh(2\pi)} = -B \tanh(2\pi)$.

$X(x) = -B \tanh(2\pi) \cosh(2\pi x) + B \sinh(2\pi x)$.

$X(x) = B \left( \sinh(2\pi x) - \tanh(2\pi) \cosh(2\pi x) \right)$.

$X(x) = B \left( \sinh(2\pi x) - \frac{\sinh(2\pi)}{\cosh(2\pi)} \cosh(2\pi x) \right)$.

$X(x) = \frac{B}{\cosh(2\pi)} \left( \sinh(2\pi x)\cosh(2\pi) - \cosh(2\pi x)\sinh(2\pi) \right)$.

Using $\sinh(A-B) = \sinh A \cosh B - \cosh A \sinh B$:

$X(x) = C \sinh(2\pi(x-1))$ where $C = B/\cosh(2\pi)$.

Question 5:

Consider $u(x,y) = \sum_{n=1}^\infty E_n \sinh(\frac{n\pi y}{a}) \sin(\frac{n\pi x}{a})$. If $u(x,b) = f(x)$, write the expression for $E_n$.

Solution:

Setting $y=b$: $f(x) = \sum_{n=1}^\infty \left[ E_n \sinh\left(\frac{n\pi b}{a}\right) \right] \sin\left(\frac{n\pi x}{a}\right)$.

The term in brackets is the $n$-th Fourier Sine coefficient of $f(x)$ on $[0, a]$.

$E_n \sinh\left(\frac{n\pi b}{a}\right) = \frac{2}{a} \int_0^a f(x) \sin\left(\frac{n\pi x}{a}\right) \, dx$.

$E_n = \frac{2}{a \sinh(n\pi b/a)} \int_0^a f(x) \sin\left(\frac{n\pi x}{a}\right) \, dx$.

Question 6:

Consider $u(x,y) = \sum_{n=1}^\infty E_n \sin(\frac{n\pi x}{a}) \sinh(\frac{n\pi (b-y)}{a})$. If $u(x,0) = f(x)$, find $E_n$.

Solution:

This form often arises when the non-homogeneous BC is at $y=0$ and $u(x,b)=0$.

Set $y=0$: $f(x) = \sum_{n=1}^\infty E_n \sin\left(\frac{n\pi x}{a}\right) \sinh\left(\frac{n\pi b}{a}\right)$.

The term $E_n \sinh(n\pi b/a)$ is the $n$-th Fourier Sine coefficient of $f(x)$ on $[0,a]$.

$E_n \sinh\left(\frac{n\pi b}{a}\right) = \frac{2}{a} \int_0^a f(x) \sin\left(\frac{n\pi x}{a}\right) \, dx$.

$E_n = \frac{2}{a \sinh(n\pi b/a)} \int_0^a f(x) \sin\left(\frac{n\pi x}{a}\right) \, dx$.

(Note: The structure ensures $u(x,b)=0$ because $\sinh(n\pi(b-b)/a) = \sinh(0) = 0$).

Question 7:

Consider $u(x,y) = \sum_{n=1}^\infty E_n \sin(\frac{n\pi y}{b}) \sinh(\frac{n\pi x}{b})$. If $u(a,y) = g(y)$, find $E_n$.

Solution:

This setup implies homogeneous BCs are $u(x,0)=0, u(x,b)=0, u(0,y)=0$. The $Y$ problem gives sines, and the $X$ problem gives sinh, satisfying $X(0)=0$.

Set $x=a$: $g(y) = \sum_{n=1}^\infty E_n \sin\left(\frac{n\pi y}{b}\right) \sinh\left(\frac{n\pi a}{b}\right)$.

The term $E_n \sinh(n\pi a/b)$ is the $n$-th Fourier Sine coefficient of $g(y)$ on $[0,b]$.

$E_n \sinh\left(\frac{n\pi a}{b}\right) = \frac{2}{b} \int_0^b g(y) \sin\left(\frac{n\pi y}{b}\right) \, dy$.

$E_n = \frac{2}{b \sinh(n\pi a/b)} \int_0^b g(y) \sin\left(\frac{n\pi y}{b}\right) \, dy$.

Question 8:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,\pi), y \in (0,1)$, with $u(0,y)=0, u(\pi,y)=0, u(x,0)=0, u(x,1)=5\sin(x)$.

Solution:

This is the standard setup from the theory with $a=\pi, b=1, f(x)=5\sin(x)$.

The separated X problem gives $X_n(x)=\sin(nx)$, $\lambda_n=n^2$.

The Y problem $Y''-n^2 Y = 0$ with $Y(0)=0$ gives $Y_n(y)=D_n \sinh(ny)$.

General solution: $u(x,y) = \sum_{n=1}^\infty E_n \sinh(ny) \sin(nx)$.

Apply $u(x,1)=5\sin(x)$: $5\sin(x) = \sum_{n=1}^\infty E_n \sinh(n\cdot 1) \sin(nx)$.

Matching coefficients for $n=1$: $E_1 \sinh(1) = 5$. $E_n=0$ for $n \neq 1$.

$E_1 = 5 / \sinh(1)$.

The solution is $u(x,y) = E_1 \sinh(1y)\sin(1x) = \frac{5}{\sinh(1)} \sinh(y)\sin(x)$.

Question 9:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,a), y \in (0,b)$, with $u(x,0)=0, u(x,b)=0, u(0,y)=0, u(a,y)=g(y)$.

Solution:

Here, the homogeneous BCs are on $y$. Assume $u=X(x)Y(y)$.

Separating: $\frac{Y''}{Y} = -\frac{X''}{X} = -\lambda$.

$Y'' + \lambda Y = 0$, $Y(0)=0, Y(b)=0$. This gives $\lambda_n=(n\pi/b)^2$, $Y_n(y)=\sin(n\pi y/b)$.

$X'' - \lambda_n X = 0 \implies X_n'' - (n\pi/b)^2 X_n = 0$.

General solution $X_n(x) = C_n \cosh(\frac{n\pi x}{b}) + D_n \sinh(\frac{n\pi x}{b})$.

Apply BC $u(0,y)=0 \implies X(0)=0$. $X_n(0) = C_n = 0$. So $X_n(x) = D_n \sinh(\frac{n\pi x}{b})$.

General solution $u(x,y) = \sum_{n=1}^\infty E_n \sinh(\frac{n\pi x}{b}) \sin(\frac{n\pi y}{b})$.

Apply $u(a,y)=g(y)$: $g(y) = \sum_{n=1}^\infty [E_n \sinh(\frac{n\pi a}{b})] \sin(\frac{n\pi y}{b})$.

$E_n \sinh(\frac{n\pi a}{b}) = \frac{2}{b} \int_0^b g(y) \sin(\frac{n\pi y}{b}) dy$.

$E_n = \frac{2}{b \sinh(n\pi a/b)} \int_0^b g(y) \sin(\frac{n\pi y}{b}) dy$.

The final solution is $u(x,y) = \sum_{n=1}^\infty \left[ \frac{2}{b \sinh(n\pi a/b)} \int_0^b g(s) \sin(\frac{n\pi s}{b}) ds \right] \sinh\left(\frac{n\pi x}{b}\right) \sin\left(\frac{n\pi y}{b}\right)$.

Question 10:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,1), y \in (0,2)$, with $u(0,y)=0, u(1,y)=0, u(x,0)=f(x), u(x,2)=0$.

Solution:

Here $a=1, b=2$. Homogeneous BCs on $x=0, x=1, y=2$. Non-homogeneous on $y=0$.

Separate: $X''+\lambda X = 0, X(0)=X(1)=0 \implies \lambda_n=(n\pi)^2, X_n=\sin(n\pi x)$.

$Y'' - \lambda_n Y = 0 \implies Y_n'' - (n\pi)^2 Y_n = 0$. $Y_n(y) = C_n \cosh(n\pi y) + D_n \sinh(n\pi y)$.

Apply $u(x,2)=0 \implies Y(2)=0$. $Y_n(2) = C_n \cosh(2n\pi) + D_n \sinh(2n\pi) = 0$.

This gives $D_n = -C_n \frac{\cosh(2n\pi)}{\sinh(2n\pi)} = -C_n \coth(2n\pi)$.

$Y_n(y) = C_n \cosh(n\pi y) - C_n \coth(2n\pi) \sinh(n\pi y)$.

$Y_n(y) = C_n \left[ \cosh(n\pi y) - \frac{\cosh(2n\pi)}{\sinh(2n\pi)} \sinh(n\pi y) \right]$.

$Y_n(y) = \frac{C_n}{\sinh(2n\pi)} \left[ \sinh(2n\pi)\cosh(n\pi y) - \cosh(2n\pi)\sinh(n\pi y) \right]$.

Using $\sinh(A-B) = \sinh A \cosh B - \cosh A \sinh B$: $A=2n\pi, B=n\pi y$.

$Y_n(y) = \frac{C_n}{\sinh(2n\pi)} \sinh(2n\pi - n\pi y) = E_n \sinh(n\pi(2-y))$. (Absorbing constant).

General solution: $u(x,y) = \sum_{n=1}^\infty E_n \sinh(n\pi(2-y)) \sin(n\pi x)$.

Apply $u(x,0)=f(x)$: $f(x) = \sum_{n=1}^\infty E_n \sinh(2n\pi) \sin(n\pi x)$.

$E_n \sinh(2n\pi) = \frac{2}{1} \int_0^1 f(x) \sin(n\pi x) dx$.

$E_n = \frac{2}{\sinh(2n\pi)} \int_0^1 f(x) \sin(n\pi x) dx$.

Final solution is $u(x,y) = \sum_{n=1}^\infty \left[ \frac{2}{\sinh(2n\pi)} \int_0^1 f(s) \sin(n\pi s) ds \right] \sinh(n\pi(2-y)) \sin(n\pi x)$.

Question 11:

Find the solution $X(x)$ to $X'' - 4X = 0$ with $X(0)=0$.

Solution:

General solution $X(x) = A\cosh(2x) + B\sinh(2x)$.

$X(0) = A(1) + B(0) = A = 0$.

So, $X(x) = B\sinh(2x)$.

Question 12:

If the boundary conditions for Laplace's equation in a rectangle are $u(0,y)=0, u(a,y)=0, u(x,0)=f(x), u(x,b)=g(x)$, how do you approach the solution using superposition?

Solution:

Let $u(x,y) = v(x,y) + w(x,y)$, where $v$ and $w$ both satisfy Laplace's equation.

Problem for $v$: $v_{xx}+v_{yy}=0$ with BCs $v(0,y)=0, v(a,y)=0, v(x,0)=f(x), v(x,b)=0$.

Problem for $w$: $w_{xx}+w_{yy}=0$ with BCs $w(0,y)=0, w(a,y)=0, w(x,0)=0, w(x,b)=g(x)$.

Solve each simpler problem using separation of variables (as shown in Q10 for $v$ and Q8/Theory for $w$). The final solution is $u(x,y) = v(x,y) + w(x,y)$.

Question 13:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,1), y \in (0,1)$, with $u(0,y)=0, u(1,y)=\sin(\pi y), u(x,0)=0, u(x,1)=0$.

Solution:

$a=1, b=1$. Homogeneous BCs on $y=0, y=1, x=0$. Non-homogeneous on $x=1$. $g(y)=\sin(\pi y)$.

Use the setup from Q9.

$Y'' + \lambda Y = 0, Y(0)=Y(1)=0 \implies \lambda_n=(n\pi)^2, Y_n=\sin(n\pi y)$.

$X'' - (n\pi)^2 X = 0, X(0)=0 \implies X_n(x)=D_n \sinh(n\pi x)$.

$u(x,y) = \sum E_n \sinh(n\pi x) \sin(n\pi y)$.

$u(1,y) = \sum E_n \sinh(n\pi) \sin(n\pi y) = \sin(\pi y)$.

Matching coefficients: $n=1$. $E_1 \sinh(\pi) = 1$. $E_n=0$ for $n\neq 1$.

$E_1 = 1/\sinh(\pi)$.

$u(x,y) = E_1 \sinh(\pi x)\sin(\pi y) = \frac{\sinh(\pi x)\sin(\pi y)}{\sinh(\pi)}$.

Question 14:

What is the general solution to $X'' = 0$?

Solution:

Integrate once: $X'(x) = A$.

Integrate again: $X(x) = Ax + B$.

Question 15:

Find the solution $Y(y)$ to $Y'' - \mu^2 Y = 0$ satisfying $Y(0)=A$ and $Y(b)=B$.

Solution:

General solution: $Y(y) = C_1 \cosh(\mu y) + C_2 \sinh(\mu y)$.

$Y(0) = C_1 \cosh(0) + C_2 \sinh(0) = C_1 = A$.

$Y(b) = A \cosh(\mu b) + C_2 \sinh(\mu b) = B$.

$C_2 \sinh(\mu b) = B - A \cosh(\mu b)$.

$C_2 = \frac{B - A \cosh(\mu b)}{\sinh(\mu b)}$ (assuming $\sinh(\mu b) \neq 0$).

$Y(y) = A \cosh(\mu y) + \frac{B - A \cosh(\mu b)}{\sinh(\mu b)} \sinh(\mu y)$.

This can be rewritten using $\sinh(A-B)$ as:

$Y(y) = \frac{A \sinh(\mu(b-y)) + B \sinh(\mu y)}{\sinh(\mu b)}$.

Question 16:

Solve $u_{xx}+u_{yy}=0$ for $x \in (0,a), y \in (0,b)$ with $u=0$ on all boundaries.

Solution:

Consider the setup from the theory (Q8). The general solution satisfying $u(0,y)=0, u(a,y)=0, u(x,0)=0$ is $u(x,y) = \sum E_n \sinh(n\pi y/a)\sin(n\pi x/a)$.

We also need $u(x,b)=0$. So, $\sum E_n \sinh(n\pi b/a)\sin(n\pi x/a) = 0$.

This is a Fourier Sine series for the zero function. All coefficients must be zero.

$E_n \sinh(n\pi b/a) = 0$. Since $\sinh(z)=0$ only if $z=0$, and $n, \pi, b, a$ are positive, $\sinh(n\pi b/a) \neq 0$.

Therefore, $E_n=0$ for all $n$.

The only solution is the trivial solution $u(x,y) = 0$.

Question 17:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,1), y \in (0,\pi)$, with $u(0,y)=0, u(1,y)=0, u(x,0)=\sin(\pi x), u(x,\pi)=0$.

Solution:

$a=1, b=\pi$. Homogeneous BCs on $x=0, x=1, y=\pi$. Non-homogeneous on $y=0$. $f(x)=\sin(\pi x)$.

Use the setup from Q10.

$X_n=\sin(n\pi x)$. $Y_n(y) = E_n \sinh(n\pi(\pi-y))$.

$u(x,y) = \sum E_n \sinh(n\pi(\pi-y))\sin(n\pi x)$.

$u(x,0) = \sum E_n \sinh(n\pi^2)\sin(n\pi x) = \sin(\pi x)$.

Matching coefficients: $n=1$. $E_1 \sinh(\pi^2) = 1$. $E_n=0$ for $n\neq 1$.

$E_1 = 1/\sinh(\pi^2)$.

$u(x,y) = E_1 \sinh(\pi(\pi-y))\sin(\pi x) = \frac{\sinh(\pi(\pi-y))\sin(\pi x)}{\sinh(\pi^2)}$.

Question 18:

What is the eigenvalue $\lambda$ if the solution to $X''+\lambda X=0$ is $X(x)=A$? What boundary conditions does this imply?

Solution:

If $X(x)=A$ (a non-zero constant), then $X'(x)=0$ and $X''(x)=0$.

Substituting into the ODE: $0 + \lambda A = 0$.

Since $A \neq 0$, we must have $\lambda = 0$.

The boundary conditions satisfied by $X(x)=A$ are Neumann conditions: $X'(0)=0$ and $X'(L)=0$.

Question 19:

Find the solution $Y(y)$ to $Y'' + 0 \cdot Y = 0$ (i.e., $Y''=0$) satisfying $Y(0)=0$.

Solution:

The general solution to $Y''=0$ is $Y(y)=Ay+B$.

Applying $Y(0)=0$: $A(0)+B=0 \implies B=0$.

The solution is $Y(y)=Ay$.

Question 20:

Solve $u_{xx}+u_{yy}=0$ for $x \in (0,1), y \in (0,1)$, $u(x,0)=0, u(x,1)=1, u(0,y)=0, u(1,y)=0$.

Solution:

$a=1, b=1$. Homogeneous on $x=0, x=1, y=0$. Non-homogeneous on $y=1$. $f(x)=1$.

Use theory setup.

$X_n=\sin(n\pi x)$, $\lambda_n=(n\pi)^2$. $Y_n(y)=E_n \sinh(n\pi y)$.

$u(x,y) = \sum E_n \sinh(n\pi y)\sin(n\pi x)$.

$u(x,1) = \sum E_n \sinh(n\pi)\sin(n\pi x) = 1$.

The coefficients $E_n \sinh(n\pi)$ are the Fourier Sine coefficients of $f(x)=1$ on $[0,1]$.

$E_n \sinh(n\pi) = \frac{2}{1} \int_0^1 1 \cdot \sin(n\pi x) dx = 2 [-\frac{\cos(n\pi x)}{n\pi}]_0^1 = -\frac{2}{n\pi}[(-1)^n-1] = \frac{2(1-(-1)^n)}{n\pi}$.

$E_n = \frac{2(1-(-1)^n)}{n\pi \sinh(n\pi)}$. $E_n=0$ for even $n$. For odd $n=2m-1$, $E_{2m-1} = \frac{4}{(2m-1)\pi \sinh((2m-1)\pi)}$.

$u(x,y) = \sum_{m=1}^\infty \frac{4}{(2m-1)\pi \sinh((2m-1)\pi)} \sinh((2m-1)\pi y) \sin((2m-1)\pi x)$.

Question 21:

Find the general solution to $Y'' + k^2 Y = 0$ where $k>0$.

Solution:

Characteristic equation $r^2+k^2=0 \implies r=\pm ik$.

General solution $Y(y) = C_1 \cos(ky) + C_2 \sin(ky)$.

Question 22:

Solve $X'' + \lambda X = 0$ with $X'(0)=0, X'(a)=0$. (Neumann BCs).

Solution:

Case 1: $\lambda=0$. $X''=0 \implies X(x)=Ax+B$. $X'(x)=A$. $X'(0)=A=0$. $X'(a)=A=0$. So $A=0$. $X(x)=B$ (constant) is a solution. Eigenvalue $\lambda_0=0$, eigenfunction $X_0(x)=1$ (or any constant).

Case 2: $\lambda=-\mu^2<0$. $X(x)=Ae^{\mu x}+Be^{-\mu x}$. $X'(x)=A\mu e^{\mu x}-B\mu e^{-\mu x}$. $X'(0)=A\mu - B\mu = 0 \implies A=B$. $X'(a)=A\mu e^{\mu a}-A\mu e^{-\mu a}=A\mu(e^{\mu a}-e^{-\mu a})=0$. Since $\mu\neq 0$, $e^{\mu a}-e^{-\mu a} \neq 0$, so $A=0$. Trivial solution.

Case 3: $\lambda=\mu^2>0$. $X(x)=A\cos(\mu x)+B\sin(\mu x)$. $X'(x)=-A\mu\sin(\mu x)+B\mu\cos(\mu x)$. $X'(0)=B\mu=0$. Since $\mu \neq 0$, $B=0$. $X(x)=A\cos(\mu x)$. $X'(a)=-A\mu\sin(\mu a)=0$. For non-trivial $A$, need $\sin(\mu a)=0$. $\mu a = n\pi$ for $n=1, 2, 3, \dots$. $\mu = n\pi/a$.

Eigenvalues $\lambda_n = (n\pi/a)^2$ for $n=1, 2, 3, \dots$. Eigenfunctions $X_n(x)=\cos(n\pi x/a)$.

Combined: $\lambda_n = (n\pi/a)^2$ for $n=0, 1, 2, \dots$, with $X_n(x) = \cos(n\pi x/a)$.

Question 23:

How does the solution form change in Q8 if the boundary condition was $u(x,1)=5\sin(3x)$ instead?

Solution:

The general solution form remains $u(x,y) = \sum E_n \sinh(ny)\sin(nx)$.

The boundary condition $u(x,1)=\sum E_n \sinh(n)\sin(nx) = 5\sin(3x)$.

Matching coefficients: $E_3 \sinh(3) = 5$. $E_n=0$ for $n\neq 3$.

$E_3 = 5/\sinh(3)$.

The solution becomes $u(x,y) = \frac{5}{\sinh(3)}\sinh(3y)\sin(3x)$. Only the index $n=3$ contributes.

Question 24:

If $u_1$ solves Laplace's equation with BCs $f_1, 0, 0, 0$ (on top, left, bottom, right) and $u_2$ solves it with BCs $0, f_2, 0, 0$, what is the solution for BCs $f_1, f_2, 0, 0$?

Solution:

By the superposition principle for linear PDEs, if $u_1$ and $u_2$ are solutions, then their sum $u = u_1+u_2$ is also a solution.

Let's check the boundary conditions for $u = u_1+u_2$:

  • Top ($y=b$): $u = u_1+u_2 = f_1+0 = f_1$.
  • Left ($x=0$): $u = u_1+u_2 = 0+f_2 = f_2$.
  • Bottom ($y=0$): $u = u_1+u_2 = 0+0 = 0$.
  • Right ($x=a$): $u = u_1+u_2 = 0+0 = 0$.

The boundary conditions match the desired problem.

Therefore, the solution is $u(x,y) = u_1(x,y) + u_2(x,y)$.

Question 25:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,\pi), y \in (0,\pi)$, with $u(x,0)=0, u(x,\pi)=\sin(2x), u(0,y)=0, u(\pi,y)=0$.

Solution:

$a=\pi, b=\pi$. Homogeneous on $x=0, x=\pi, y=0$. Non-homogeneous on $y=\pi$. $f(x)=\sin(2x)$.

Use theory setup.

$X_n=\sin(nx)$, $\lambda_n=n^2$. $Y_n(y)=E_n \sinh(ny)$.

$u(x,y) = \sum E_n \sinh(ny)\sin(nx)$.

$u(x,\pi) = \sum E_n \sinh(n\pi)\sin(nx) = \sin(2x)$.

Matching: $n=2$. $E_2 \sinh(2\pi) = 1$. $E_n=0$ for $n\neq 2$.

$E_2 = 1/\sinh(2\pi)$.

$u(x,y) = E_2 \sinh(2y)\sin(2x) = \frac{\sinh(2y)\sin(2x)}{\sinh(2\pi)}$.

Question 26:

What kind of functions (trigonometric or hyperbolic) typically arise for the variable corresponding to the sides with homogeneous boundary conditions in Laplace's equation?

Solution:

The variable corresponding to the homogeneous boundary conditions (e.g., $X(x)$ if $u(0,y)=u(a,y)=0$) leads to an eigenvalue problem like $X''+\lambda X=0$ with boundary conditions.

The non-trivial solutions to such eigenvalue problems with standard homogeneous boundary conditions (Dirichlet, Neumann, Robin, Periodic) are typically trigonometric functions (sines and/or cosines).

Question 27:

What kind of functions (trigonometric or hyperbolic) typically arise for the variable corresponding to the sides with non-homogeneous boundary conditions in Laplace's equation?

Solution:

The ODE for the other variable (e.g., $Y(y)$ if $X(x)$ had homogeneous BCs) typically takes the form $Y'' - \lambda_n Y = 0$, where $\lambda_n$ are the positive eigenvalues found from the $X$ problem.

The solutions to $Y'' - \mu^2 Y = 0$ (where $\mu^2 = \lambda_n > 0$) are exponential functions $e^{\mu y}, e^{-\mu y}$ or, more commonly used, hyperbolic functions $\sinh(\mu y), \cosh(\mu y)$.

Question 28:

Solve $u_{xx} + u_{yy} = 0$ for $x \in (0,1), y \in (0,1)$, with $u(0,y)=\sin(2\pi y), u(1,y)=0, u(x,0)=0, u(x,1)=0$.

Solution:

$a=1, b=1$. Homogeneous on $x=1, y=0, y=1$. Non-homogeneous on $x=0$. $g(y)=\sin(2\pi y)$.

Separate: $Y''+\lambda Y=0, Y(0)=Y(1)=0 \implies \lambda_n=(n\pi)^2, Y_n=\sin(n\pi y)$.

$X'' - (n\pi)^2 X = 0$. $X_n(x) = C_n \cosh(n\pi x) + D_n \sinh(n\pi x)$.

Apply $u(1,y)=0 \implies X(1)=0$. $X_n(1) = C_n \cosh(n\pi) + D_n \sinh(n\pi) = 0$.

As in Q4, this leads to solutions of the form $X_n(x)=E_n \sinh(n\pi(1-x))$.

$u(x,y) = \sum E_n \sinh(n\pi(1-x))\sin(n\pi y)$.

$u(0,y) = \sum E_n \sinh(n\pi)\sin(n\pi y) = \sin(2\pi y)$.

Matching: $n=2$. $E_2 \sinh(2\pi) = 1$. $E_n=0$ for $n\neq 2$.

$E_2=1/\sinh(2\pi)$.

$u(x,y) = \frac{\sinh(2\pi(1-x))\sin(2\pi y)}{\sinh(2\pi)}$.

Question 29:

Find the coefficient $E_1$ in Q8 if $f(x)=x$. ($a=\pi, b=1$).

Solution:

From Q8 setup, $u(x,y) = \sum E_n \sinh(ny)\sin(nx)$.

$u(x,1) = \sum E_n \sinh(n)\sin(nx) = x$.

$E_n \sinh(n)$ are the Fourier Sine coefficients of $f(x)=x$ on $[0, \pi]$.

$E_n \sinh(n) = \frac{2}{\pi} \int_0^\pi x \sin(nx) dx = \frac{2(-1)^{n+1}}{n}$. (From Q17/Q20).

For $n=1$: $E_1 \sinh(1) = \frac{2(-1)^{1+1}}{1} = 2$.

$E_1 = 2/\sinh(1)$.

Question 30:

Why can we use superposition for Laplace's equation with boundary conditions split across different sides?

Solution:

Laplace's equation ($u_{xx}+u_{yy}=0$) is a linear and homogeneous PDE. The boundary conditions are also typically linear.

Linearity means that if $u_1$ and $u_2$ are solutions to the PDE, then any linear combination $c_1 u_1 + c_2 u_2$ is also a solution to the PDE itself.

When splitting the problem, say $u=v+w$, both $v$ and $w$ satisfy Laplace's equation, so their sum $u$ also satisfies it.

The boundary conditions are additive. If $v$ satisfies the BC on side 1 and $w$ satisfies the BC on side 2 (with both satisfying 0 on the other sides), then $u=v+w$ will satisfy the combined BCs on sides 1 and 2. For example, on side 1: $u=v+w = f_1+0 = f_1$. On side 2: $u=v+w=0+f_2=f_2$. On other sides: $u=v+w=0+0=0$.

This allows us to break down a complex BVP into simpler BVPs that we can solve individually using techniques like separation of variables, and then add the results.

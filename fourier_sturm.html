Sturm-Liouville Theory & Fourier Series (Full/Cosine)

Theory Focus

Sturm-Liouville Problems

A **regular Sturm-Liouville (S-L) problem** on an interval $[a, b]$ involves finding eigenvalues $\lambda$ and non-trivial eigenfunctions $y(x)$ that satisfy:

  1. The differential equation: $\frac{d}{dx}\left[p(x)\frac{dy}{dx}\right] + q(x)y + \lambda r(x)y = 0$
  2. Separated boundary conditions: $\alpha_1 y(a) + \alpha_2 y'(a) = 0$
    $\beta_1 y(b) + \beta_2 y'(b) = 0$

Where $p(x)>0, r(x)>0$ are continuously differentiable, $q(x)$ is continuous, and $\alpha_1, \alpha_2$ (and $\beta_1, \beta_2$) are not both zero.

Key Properties:

  • There exists an infinite sequence of real eigenvalues $\lambda_1 < \lambda_2 < \dots$ with $\lambda_n \to \infty$.
  • Corresponding to each eigenvalue $\lambda_n$ is an eigenfunction $y_n(x)$ (unique up to a constant multiple).
  • The eigenfunctions $\{y_n(x)\}_{n=1}^\infty$ form an **orthogonal set** with respect to the **weight function $r(x)$** on $[a, b]$: $$ \int_a^b y_n(x) y_m(x) r(x) \, dx = 0 \quad \text{if } n \neq m $$
  • This set of eigenfunctions forms a complete basis, meaning "nice" functions $f(x)$ defined on $[a, b]$ can be represented by a **generalized Fourier series**: $$ f(x) = \sum_{n=1}^\infty c_n y_n(x) $$ where the coefficients are found using orthogonality: $$ c_n = \frac{\int_a^b f(x) y_n(x) r(x) \, dx}{\int_a^b [y_n(x)]^2 r(x) \, dx} $$ The denominator is the squared norm $\|y_n\|^2 = \int_a^b y_n^2 r dx$.

Relevance to PDEs: Separation of variables $u=X(x)T(t)$ often leads to a Sturm-Liouville problem for the spatial part $X(x)$. The type of boundary conditions on $u$ dictates the boundary conditions for $X(x)$.

Standard Fourier Series

These arise from the S-L problem $y'' + \lambda y = 0$ (where $p(x)=1, q(x)=0, r(x)=1$) with different boundary conditions.

1. Full Fourier Series on $[-L, L]$

Associated with periodic boundary conditions $y(-L)=y(L), y'(-L)=y'(L)$.

Any "nice" function $f(x)$ defined on $[-L, L]$ can be represented as:

$$ f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty \left[ a_n \cos\left(\frac{n\pi x}{L}\right) + b_n \sin\left(\frac{n\pi x}{L}\right) \right] $$

Coefficients:

$$ a_0 = \frac{1}{L} \int_{-L}^L f(x) \, dx $$ $$ a_n = \frac{1}{L} \int_{-L}^L f(x) \cos\left(\frac{n\pi x}{L}\right) \, dx \quad (n=1, 2, \dots) $$ $$ b_n = \frac{1}{L} \int_{-L}^L f(x) \sin\left(\frac{n\pi x}{L}\right) \, dx \quad (n=1, 2, \dots) $$

Note: If $f(x)$ is even, all $b_n=0$. If $f(x)$ is odd, all $a_n=0$ (including $a_0$).

2. Fourier Cosine Series on $[0, L]$

Associated with Neumann boundary conditions $y'(0)=0, y'(L)=0$. The S-L problem $X''+\lambda X=0, X'(0)=X'(L)=0$ has eigenvalues $\lambda_n=(n\pi/L)^2$ for $n=0,1,2,\dots$ and eigenfunctions $X_n(x)=\cos(n\pi x/L)$.

Any "nice" function $f(x)$ defined on $[0, L]$ can be represented as:

$$ f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty a_n \cos\left(\frac{n\pi x}{L}\right) $$

Coefficients:

$$ a_0 = \frac{2}{L} \int_{0}^L f(x) \, dx $$ $$ a_n = \frac{2}{L} \int_{0}^L f(x) \cos\left(\frac{n\pi x}{L}\right) \, dx \quad (n=1, 2, \dots) $$

This series corresponds to the **even extension** of $f(x)$ to $[-L, L]$.

3. Fourier Sine Series on $[0, L]$ (Recap from Day 4)

Associated with Dirichlet boundary conditions $y(0)=0, y(L)=0$. The S-L problem $X''+\lambda X=0, X(0)=X(L)=0$ has eigenvalues $\lambda_n=(n\pi/L)^2$ for $n=1,2,3,\dots$ and eigenfunctions $X_n(x)=\sin(n\pi x/L)$.

Any "nice" function $f(x)$ defined on $[0, L]$ can be represented as:

$$ f(x) = \sum_{n=1}^\infty b_n \sin\left(\frac{n\pi x}{L}\right) $$

Coefficients:

$$ b_n = \frac{2}{L} \int_{0}^L f(x) \sin\left(\frac{n\pi x}{L}\right) \, dx \quad (n=1, 2, \dots) $$

This series corresponds to the **odd extension** of $f(x)$ to $[-L, L]$.

Practice Questions

Question 1:

Identify $p(x), q(x), r(x)$ for the S-L problem $y'' + \lambda y = 0$.

Solution:

Rewrite as $(1 \cdot y')' + 0 \cdot y + \lambda \cdot 1 \cdot y = 0$.

Comparing with $(p y')' + qy + \lambda r y = 0$, we have:

$p(x)=1$, $q(x)=0$, $r(x)=1$.

Question 2:

Identify $p(x), q(x), r(x)$ for the S-L problem $(x y')' + \frac{1}{x} y + \lambda x y = 0$.

Solution:

Comparing with $(p y')' + qy + \lambda r y = 0$, we have:

$p(x)=x$, $q(x)=1/x$, $r(x)=x$.

Question 3:

Verify the orthogonality of $y_1=\cos(\pi x/L)$ and $y_2=\cos(2\pi x/L)$ on $[0, L]$ with weight $r(x)=1$.

Solution:

We need to calculate $\int_0^L y_1(x) y_2(x) r(x) dx = \int_0^L \cos(\frac{\pi x}{L}) \cos(\frac{2\pi x}{L}) \cdot 1 \, dx$.

Use product-to-sum: $\cos A \cos B = \frac{1}{2}[\cos(A-B) + \cos(A+B)]$.

$\int_0^L \frac{1}{2}[\cos(\frac{\pi x}{L} - \frac{2\pi x}{L}) + \cos(\frac{\pi x}{L} + \frac{2\pi x}{L})] dx = \frac{1}{2} \int_0^L [\cos(-\frac{\pi x}{L}) + \cos(\frac{3\pi x}{L})] dx$.

$= \frac{1}{2} \int_0^L [\cos(\frac{\pi x}{L}) + \cos(\frac{3\pi x}{L})] dx$.

$= \frac{1}{2} \left[ \frac{L}{\pi}\sin(\frac{\pi x}{L}) + \frac{L}{3\pi}\sin(\frac{3\pi x}{L}) \right]_0^L$.

$= \frac{1}{2} [ (\frac{L}{\pi}\sin(\pi) + \frac{L}{3\pi}\sin(3\pi)) - (\frac{L}{\pi}\sin(0) + \frac{L}{3\pi}\sin(0)) ]$.

$= \frac{1}{2} [ (0+0) - (0+0) ] = 0$.

The functions are orthogonal.

Question 4:

Find the full Fourier Series for $f(x) = x$ on $[-\pi, \pi]$.

Solution:

$L=\pi$. Since $f(x)=x$ is an odd function, we expect $a_n=0$ for all $n \ge 0$.

$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} x dx = \frac{1}{\pi} [\frac{x^2}{2}]_{-\pi}^{\pi} = \frac{1}{\pi}[\frac{\pi^2}{2}-\frac{(-\pi)^2}{2}]=0$.

$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \cos(nx) dx$. Since $x\cos(nx)$ is odd, the integral over a symmetric interval is 0. $a_n=0$.

$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \sin(nx) dx$. Since $x\sin(nx)$ is even, we can write:

$b_n = \frac{2}{\pi} \int_{0}^{\pi} x \sin(nx) dx$.

Using integration by parts ($u=x, dv=\sin(nx)dx$):

$b_n = \frac{2}{\pi} \left[ -x\frac{\cos(nx)}{n} \Big|_0^\pi - \int_0^\pi -\frac{\cos(nx)}{n} dx \right]$.

$b_n = \frac{2}{\pi} \left[ -\pi\frac{\cos(n\pi)}{n} - 0 + \frac{1}{n} \int_0^\pi \cos(nx) dx \right]$.

$b_n = \frac{2}{\pi} \left[ -\pi\frac{(-1)^n}{n} + \frac{1}{n} [\frac{\sin(nx)}{n}]_0^\pi \right]$.

$b_n = \frac{2}{\pi} \left[ \frac{\pi(-1)^{n+1}}{n} + \frac{1}{n^2}(0-0) \right] = \frac{2(-1)^{n+1}}{n}$.

The series is $f(x) = \sum_{n=1}^\infty \frac{2(-1)^{n+1}}{n} \sin(nx)$.

Question 5:

Find the Fourier Cosine Series for $f(x) = x$ on $[0, \pi]$.

Solution:

$L=\pi$. $f(x) = \frac{a_0}{2} + \sum a_n \cos(nx)$.

$a_0 = \frac{2}{\pi} \int_0^\pi x dx = \frac{2}{\pi} [\frac{x^2}{2}]_0^\pi = \frac{2}{\pi} \frac{\pi^2}{2} = \pi$.

$a_n = \frac{2}{\pi} \int_0^\pi x \cos(nx) dx$. Use integration by parts ($u=x, dv=\cos(nx)dx$).

$a_n = \frac{2}{\pi} \left[ x\frac{\sin(nx)}{n} \Big|_0^\pi - \int_0^\pi \frac{\sin(nx)}{n} dx \right]$.

$a_n = \frac{2}{\pi} \left[ (\pi\frac{\sin(n\pi)}{n} - 0) - \frac{1}{n} [-\frac{\cos(nx)}{n}]_0^\pi \right]$.

$a_n = \frac{2}{\pi} \left[ 0 + \frac{1}{n^2} (\cos(n\pi)-\cos(0)) \right] = \frac{2}{\pi n^2} ((-1)^n - 1)$.

$a_n = 0$ for even $n$. For odd $n=2m-1$, $a_{2m-1} = \frac{2}{\pi(2m-1)^2}(-1-1) = -\frac{4}{\pi(2m-1)^2}$.

The series is $f(x) = \frac{\pi}{2} + \sum_{m=1}^\infty \frac{-4}{\pi(2m-1)^2} \cos((2m-1)x)$.

Question 6:

Find the Full Fourier Series for $f(x) = |x|$ on $[-\pi, \pi]$.

Solution:

$L=\pi$. $f(x)=|x|$ is an even function, so we expect $b_n=0$.

$a_0 = \frac{1}{\pi} \int_{-\pi}^\pi |x| dx = \frac{2}{\pi} \int_0^\pi x dx = \frac{2}{\pi} [\frac{x^2}{2}]_0^\pi = \frac{2}{\pi}\frac{\pi^2}{2} = \pi$.

$a_n = \frac{1}{\pi} \int_{-\pi}^\pi |x| \cos(nx) dx = \frac{2}{\pi} \int_0^\pi x \cos(nx) dx$.

From Q5, this integral evaluates to $\frac{2}{\pi n^2}((-1)^n-1)$.

So $a_n = \frac{2}{\pi} \cdot \frac{2}{\pi n^2}((-1)^n-1) = \frac{4}{\pi n^2}((-1)^n-1)$. Wait, Q5 had $\frac{2}{\pi}\int...$

Let's re-evaluate $a_n$ from Q5. $a_n = \frac{2}{\pi} \int_0^\pi x \cos(nx) dx = \frac{2}{\pi} [\frac{1}{n^2}((-1)^n-1)]$. This is correct.

$a_n = \frac{2}{\pi n^2}((-1)^n-1)$.

$a_n=0$ for even $n$. For odd $n=2m-1$, $a_{2m-1} = \frac{2}{\pi(2m-1)^2}(-1-1) = -\frac{4}{\pi(2m-1)^2}$.

$b_n = \frac{1}{\pi} \int_{-\pi}^\pi |x| \sin(nx) dx$. Since $|x|\sin(nx)$ is odd, $b_n=0$.

The series is $f(x) = \frac{a_0}{2} + \sum a_n \cos(nx) = \frac{\pi}{2} + \sum_{m=1}^\infty \frac{-4}{\pi(2m-1)^2} \cos((2m-1)x)$.

(Note: This is identical to the Fourier Cosine series of $f(x)=x$ on $[0, \pi]$, as expected, since $|x|$ is the even extension of $x$).

Question 7:

Find the Fourier Cosine Series for $f(x)=1$ on $[0, L]$.

Solution:

$f(x) = \frac{a_0}{2} + \sum a_n \cos(n\pi x/L)$.

$a_0 = \frac{2}{L} \int_0^L 1 dx = \frac{2}{L} [x]_0^L = \frac{2}{L}(L) = 2$.

$a_n = \frac{2}{L} \int_0^L 1 \cdot \cos(\frac{n\pi x}{L}) dx = \frac{2}{L} [\frac{L}{n\pi} \sin(\frac{n\pi x}{L})]_0^L$.

$a_n = \frac{2}{n\pi} [\sin(n\pi) - \sin(0)] = \frac{2}{n\pi}[0-0] = 0$ for $n \ge 1$.

The series is $f(x) = \frac{a_0}{2} + 0 = \frac{2}{2} = 1$.

The Fourier Cosine Series for a constant is just the constant itself.

Question 8:

Find the Full Fourier Series for $f(x)=\sin x$ on $[-\pi, \pi]$.

Solution:

$L=\pi$. $f(x)=\sin x$ is an odd function, so $a_n=0$ for all $n \ge 0$.

$b_n = \frac{1}{\pi} \int_{-\pi}^\pi \sin x \sin(nx) dx = \frac{2}{\pi} \int_0^\pi \sin x \sin(nx) dx$.

By orthogonality of sines on $[0, \pi]$, this integral is 0 if $n \neq 1$, and $\pi/2$ if $n=1$.

So, $b_1 = \frac{2}{\pi} (\frac{\pi}{2}) = 1$. $b_n = 0$ for $n \neq 1$.

The series is $f(x) = 0 + \sum [0 + b_n \sin(nx)] = b_1 \sin(1x) = \sin x$.

Question 9:

Which type of Fourier Series (Sine, Cosine, Full) would you use to represent the initial condition $u(x,0)=f(x)$ for a PDE solved on $[0, L]$ with boundary conditions $u_x(0,t)=0, u_x(L,t)=0$?

Solution:

The Neumann boundary conditions $u_x(0,t)=0, u_x(L,t)=0$ lead to the spatial eigenvalue problem $X''+\lambda X=0$ with $X'(0)=0, X'(L)=0$.

The eigenfunctions for this problem are $X_n(x)=\cos(n\pi x/L)$ (including $n=0$).

Therefore, the solution $u(x,t)$ will be a superposition of these cosine functions. To match the initial condition $u(x,0)=f(x)$, we need to represent $f(x)$ as a series of these cosine eigenfunctions, which is the **Fourier Cosine Series**.

Question 10:

Find the Fourier Cosine Series coefficient $a_0$ for $f(x)=x^2$ on $[0, \pi]$.

Solution:

$L=\pi$. $a_0 = \frac{2}{\pi} \int_0^\pi f(x) dx = \frac{2}{\pi} \int_0^\pi x^2 dx$.

$a_0 = \frac{2}{\pi} [\frac{x^3}{3}]_0^\pi = \frac{2}{\pi} [\frac{\pi^3}{3} - 0] = \frac{2\pi^2}{3}$.

($\frac{a_0}{2} = \frac{\pi^2}{3}$).

Question 11:

Find the Fourier Sine Series coefficient $b_2$ for $f(x)=x$ on $[0, \pi]$.

Solution:

$L=\pi$. $b_n = \frac{2(-1)^{n+1}}{n}$.

For $n=2$, $b_2 = \frac{2(-1)^{2+1}}{2} = \frac{2(-1)}{2} = -1$.

Question 12:

Find the Full Fourier Series for $f(x)=1+\cos x$ on $[-\pi, \pi]$.

Solution:

$f(x)$ is even, so $b_n=0$. $L=\pi$.

$a_0 = \frac{1}{\pi}\int_{-\pi}^\pi (1+\cos x) dx = \frac{2}{\pi}\int_0^\pi (1+\cos x) dx = \frac{2}{\pi}[x+\sin x]_0^\pi = \frac{2}{\pi}[\pi+0-0]=2$.

$a_n = \frac{1}{\pi}\int_{-\pi}^\pi (1+\cos x)\cos(nx) dx = \frac{2}{\pi}\int_0^\pi (\cos(nx)+\cos x \cos(nx)) dx$.

$\int_0^\pi \cos(nx) dx = [\frac{\sin(nx)}{n}]_0^\pi = 0$ for $n\ge 1$.

$\int_0^\pi \cos x \cos(nx) dx$. By orthogonality, this is 0 if $n\neq 1$. If $n=1$, the integral is $\int_0^\pi \cos^2 x dx = \int_0^\pi \frac{1+\cos(2x)}{2} dx = \frac{1}{2}[x+\frac{\sin(2x)}{2}]_0^\pi = \frac{1}{2}[\pi-0]=\pi/2$.

So, $a_1 = \frac{2}{\pi}(0 + \pi/2) = 1$. $a_n = \frac{2}{\pi}(0+0)=0$ for $n \ge 2$.

Series is $f(x) = \frac{a_0}{2} + a_1\cos(1x) = \frac{2}{2} + 1\cos x = 1+\cos x$. (The function is its own Fourier series).

Question 13:

To what value does the Fourier Sine series of $f(x)=x$ on $[0, \pi]$ converge at $x=\pi$?

Solution:

The Fourier Sine series corresponds to the odd periodic extension of $f(x)$. The odd extension $f_{odd}(x)=x$ on $(-\pi, \pi)$ has jumps at $x=\pm \pi$. The odd extension is periodic with period $2\pi$.

$f_{odd}(\pi^-) = \pi$. $f_{odd}(\pi^+) = f_{odd}(-\pi^+) = -\pi$.

At a jump discontinuity, the Fourier series converges to the average of the left and right limits.

Convergence at $x=\pi$: $\frac{f_{odd}(\pi^-) + f_{odd}(\pi^+)}{2} = \frac{\pi + (-\pi)}{2} = 0$.

Question 14:

To what value does the Fourier Cosine series of $f(x)=x$ on $[0, \pi]$ converge at $x=\pi$?

Solution:

The Fourier Cosine series corresponds to the even periodic extension $f_{even}(x)=|x|$ on $[-\pi, \pi]$. This extension is continuous everywhere.

The Fourier series of a continuous function converges to the function value itself.

At $x=\pi$, the value of the even extension is $f_{even}(\pi)=|\pi|=\pi$.

The series converges to $\pi$.

Question 15:

Find the Fourier Sine Series coefficient $b_1$ for $f(x)=1$ on $[0, \pi]$.

Solution:

$L=\pi$. $b_n = -\frac{2}{n\pi}[(-1)^n - 1]$.

For $n=1$, $b_1 = -\frac{2}{1\pi}[(-1)^1 - 1] = -\frac{2}{\pi}[-1 - 1] = -\frac{2}{\pi}[-2] = \frac{4}{\pi}$.

Question 16:

Identify $p, q, r$ for Legendre's equation $(1-x^2)y'' - 2xy' + \lambda y = 0$ in S-L form.

Solution:

We need to write it as $(p y')' + qy + \lambda r y = 0$.

Divide by $(1-x^2)$: $y'' - \frac{2x}{1-x^2}y' + \frac{\lambda}{1-x^2} y = 0$.

The integrating factor for the first two terms is $e^{\int -\frac{2x}{1-x^2} dx}$. Let $v=1-x^2, dv=-2x dx$. Integral is $e^{\int \frac{dv}{v}} = e^{\ln|1-x^2|} = |1-x^2|$. For $x \in (-1, 1)$, this is $1-x^2$.

Multiply the original equation by $\frac{1}{1-x^2}$ (no, multiply the *standard* form by $p$).

Try to recognize $(p y')'$. Note that $((1-x^2)y')' = (1-x^2)y'' + (-2x)y'$. This matches the first two terms.

So the equation is $((1-x^2)y')' + \lambda \cdot 1 \cdot y = 0$.

$p(x) = 1-x^2$, $q(x) = 0$, $r(x) = 1$. (Also note $\lambda$ here is $n(n+1)$ usually).

Question 17:

Find the Fourier Cosine Series coefficient $a_1$ for $f(x)=\sin x$ on $[0, \pi]$.

Solution:

$L=\pi$. $a_n = \frac{2}{\pi} \int_0^\pi f(x)\cos(nx) dx$.

$a_1 = \frac{2}{\pi} \int_0^\pi \sin x \cos(1x) dx = \frac{2}{\pi} \int_0^\pi \frac{1}{2}\sin(2x) dx$.

$a_1 = \frac{1}{\pi} [-\frac{\cos(2x)}{2}]_0^\pi = -\frac{1}{2\pi}[\cos(2\pi)-\cos(0)] = -\frac{1}{2\pi}[1-1]=0$.

Question 18:

Find the Full Fourier Series coefficient $b_2$ for $f(x)=x^2$ on $[-\pi, \pi]$.

Solution:

$f(x)=x^2$ is an even function. Therefore, all sine coefficients $b_n$ must be 0.

$b_2 = 0$.

Question 19:

What boundary conditions correspond to the Fourier Sine Series on $[0, L]$?

Solution:

The Fourier Sine Series uses basis functions $y_n(x)=\sin(n\pi x/L)$.

These functions satisfy $y_n(0)=0$ and $y_n(L)=0$.

These correspond to **Dirichlet** boundary conditions.

Question 20:

Find the Fourier Cosine Series coefficient $a_2$ for $f(x)=x$ on $[0, \pi]$.

Solution:

$L=\pi$. From Q5, $a_n = \frac{2}{\pi n^2}((-1)^n-1)$.

For $n=2$, $a_2 = \frac{2}{\pi (2^2)}((-1)^2-1) = \frac{2}{4\pi}(1-1) = 0$.

Question 21:

What is the value of $\int_0^1 \sin(2\pi x)\sin(3\pi x) dx$?

Solution:

This involves eigenfunctions $\sin(n\pi x/L)$ with $L=1$ for $n=2$ and $n=3$.

Since the eigenfunctions are orthogonal on $[0,L]$ with weight $r=1$, the integral $\int_0^L \sin(n\pi x/L)\sin(m\pi x/L)dx = 0$ if $n\neq m$.

Here $n=2, m=3$, so the integral is 0.

Question 22:

Find the Full Fourier Series for $f(x)=3$ on $[-2, 2]$.

Solution:

$L=2$. $f(x)=3$ is even, so $b_n=0$.

$a_0 = \frac{1}{L}\int_{-L}^L f(x) dx = \frac{1}{2}\int_{-2}^2 3 dx = \frac{3}{2}[x]_{-2}^2 = \frac{3}{2}(2-(-2)) = \frac{3}{2}(4)=6$.

$a_n = \frac{1}{L}\int_{-L}^L f(x)\cos(\frac{n\pi x}{L}) dx = \frac{1}{2}\int_{-2}^2 3\cos(\frac{n\pi x}{2}) dx = \frac{3}{2} [\frac{2}{n\pi}\sin(\frac{n\pi x}{2})]_{-2}^2$.

$a_n = \frac{3}{n\pi}[\sin(n\pi)-\sin(-n\pi)] = \frac{3}{n\pi}[0-0]=0$ for $n \ge 1$.

Series is $f(x) = \frac{a_0}{2} = \frac{6}{2} = 3$.

Question 23:

Find the Fourier Cosine Series for $f(x)=\sin x$ on $[0, \pi]$. Find $a_2$.

Solution:

$L=\pi$. $a_n = \frac{2}{\pi} \int_0^\pi \sin x \cos(nx) dx$.

Use product-to-sum: $\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]$.

$a_n = \frac{2}{\pi} \int_0^\pi \frac{1}{2}[\sin(x+nx) + \sin(x-nx)] dx = \frac{1}{\pi} \int_0^\pi [\sin((n+1)x) - \sin((n-1)x)] dx$.

If $n=1$, $a_1 = \frac{1}{\pi} \int_0^\pi [\sin(2x) - \sin(0)] dx = \frac{1}{\pi} [-\frac{\cos(2x)}{2}]_0^\pi = -\frac{1}{2\pi}[1-1]=0$. (Matches Q17).

If $n \neq 1$:

$a_n = \frac{1}{\pi} [-\frac{\cos((n+1)x)}{n+1} + \frac{\cos((n-1)x)}{n-1}]_0^\pi$.

$a_n = \frac{1}{\pi} [ (-\frac{\cos((n+1)\pi)}{n+1} + \frac{\cos((n-1)\pi)}{n-1}) - (-\frac{1}{n+1} + \frac{1}{n-1}) ]$.

$a_n = \frac{1}{\pi} [ -\frac{(-1)^{n+1}}{n+1} + \frac{(-1)^{n-1}}{n-1} + \frac{1}{n+1} - \frac{1}{n-1} ]$.

Note $(-1)^{n+1}=(-1)^{n-1}$. Let $P=(-1)^{n-1}$.

$a_n = \frac{1}{\pi} [ \frac{P}{n-1} - \frac{P}{n+1} + \frac{1}{n+1} - \frac{1}{n-1} ] = \frac{1}{\pi} [ (P-1)(\frac{1}{n-1}-\frac{1}{n+1}) ]$.

$a_n = \frac{P-1}{\pi} \frac{(n+1)-(n-1)}{(n-1)(n+1)} = \frac{P-1}{\pi} \frac{2}{n^2-1}$.

If $n$ is odd ($n \ge 3$), $n-1$ is even, $P=(-1)^{even}=1$. $a_n = \frac{1-1}{\pi}\dots = 0$.

If $n$ is even ($n \ge 2$), $n-1$ is odd, $P=(-1)^{odd}=-1$. $a_n = \frac{-1-1}{\pi}\frac{2}{n^2-1} = \frac{-4}{\pi(n^2-1)}$.

We need $a_2$. $n=2$ is even. $a_2 = \frac{-4}{\pi(2^2-1)} = \frac{-4}{3\pi}$.

Question 24:

Find the norm squared $\|y_n\|^2 = \int_0^L y_n^2 r dx$ for the eigenfunctions $y_n=\sin(n\pi x/L)$ of $y''+\lambda y=0, y(0)=y(L)=0$.

Solution:

Here $r(x)=1$. We need $\|y_n\|^2 = \int_0^L \sin^2(\frac{n\pi x}{L}) dx$.

Using $\sin^2\theta = (1-\cos(2\theta))/2$:

$\|y_n\|^2 = \int_0^L \frac{1-\cos(2n\pi x/L)}{2} dx = \frac{1}{2}[x - \frac{L}{2n\pi}\sin(\frac{2n\pi x}{L})]_0^L$.

$\|y_n\|^2 = \frac{1}{2}[(L-0)-(0-0)] = \frac{L}{2}$.

Question 25:

Find the norm squared $\|y_n\|^2$ for the eigenfunctions $y_n=\cos(n\pi x/L)$ ($n \ge 1$) of $y''+\lambda y=0, y'(0)=y'(L)=0$.

Solution:

$r(x)=1$. $\|y_n\|^2 = \int_0^L \cos^2(\frac{n\pi x}{L}) dx$.

Using $\cos^2\theta = (1+\cos(2\theta))/2$:

$\|y_n\|^2 = \int_0^L \frac{1+\cos(2n\pi x/L)}{2} dx = \frac{1}{2}[x + \frac{L}{2n\pi}\sin(\frac{2n\pi x}{L})]_0^L$.

$\|y_n\|^2 = \frac{1}{2}[(L+0)-(0+0)] = \frac{L}{2}$ (for $n \ge 1$).

Question 26:

What is the norm squared $\|y_0\|^2$ for the eigenfunction $y_0=1$ ($\lambda_0=0$) in Q25?

Solution:

$\|y_0\|^2 = \int_0^L y_0^2 r dx = \int_0^L (1)^2 \cdot 1 dx = \int_0^L 1 dx = [x]_0^L = L$.

(This is why the $a_0$ term has a $1/L$ or $2/L$ factor while $a_n, b_n$ have $2/L$ from $1/(L/2)$).

Question 27:

If $f(x)$ on $[0,L]$ is represented by a Sine series, what is the value of the series at $x=0$ and $x=L$?

Solution:

The Sine series is $f(x) = \sum b_n \sin(n\pi x/L)$.

At $x=0$, $\sin(0)=0$, so the series converges to 0.

At $x=L$, $\sin(n\pi L/L)=\sin(n\pi)=0$, so the series converges to 0.

This matches the Dirichlet boundary conditions $u(0,t)=0, u(L,t)=0$ for which the Sine series is used.

Question 28:

Find the Full Fourier Series for $f(x)=e^x$ on $[-1, 1]$. Write the formula for $a_n$.

Solution:

$L=1$. $f(x)$ is neither even nor odd.

$a_n = \frac{1}{L} \int_{-L}^L f(x)\cos(\frac{n\pi x}{L}) dx = \frac{1}{1} \int_{-1}^1 e^x \cos(n\pi x) dx$.

This requires integration by parts twice.

Let $I = \int e^x \cos(n\pi x) dx$.

$u=e^x, dv=\cos(n\pi x)dx \implies du=e^x dx, v=\frac{1}{n\pi}\sin(n\pi x)$.

$I = \frac{e^x}{n\pi}\sin(n\pi x) - \int \frac{e^x}{n\pi}\sin(n\pi x) dx$.

Second integral: $u=e^x, dv=\sin(n\pi x)dx \implies du=e^x dx, v=-\frac{1}{n\pi}\cos(n\pi x)$.

$\int e^x \sin(n\pi x) dx = -\frac{e^x}{n\pi}\cos(n\pi x) - \int (-\frac{e^x}{n\pi}\cos(n\pi x)) dx = -\frac{e^x}{n\pi}\cos(n\pi x) + \frac{1}{n\pi} I$.

$I = \frac{e^x}{n\pi}\sin(n\pi x) - \frac{1}{n\pi} [-\frac{e^x}{n\pi}\cos(n\pi x) + \frac{1}{n\pi} I]$.

$I = \frac{e^x}{n\pi}\sin(n\pi x) + \frac{e^x}{(n\pi)^2}\cos(n\pi x) - \frac{1}{(n\pi)^2} I$.

$I(1 + \frac{1}{(n\pi)^2}) = \frac{e^x}{(n\pi)^2} [n\pi \sin(n\pi x) + \cos(n\pi x)]$.

$I \frac{(n\pi)^2+1}{(n\pi)^2} = \frac{e^x}{(n\pi)^2} [\dots]$.

$I = \int e^x \cos(n\pi x) dx = \frac{e^x}{(n\pi)^2+1} [n\pi \sin(n\pi x) + \cos(n\pi x)]$.

$a_n = [I]_{-1}^1 = \frac{e^1}{(n\pi)^2+1}[n\pi \sin(n\pi)+\cos(n\pi)] - \frac{e^{-1}}{(n\pi)^2+1}[n\pi \sin(-n\pi)+\cos(-n\pi)]$.

$a_n = \frac{1}{(n\pi)^2+1} [e(0+(-1)^n) - e^{-1}(0+(-1)^n)] = \frac{(-1)^n (e-e^{-1})}{(n\pi)^2+1} = \frac{2(-1)^n \sinh(1)}{(n\pi)^2+1}$.

Question 29:

Find $a_0$ for the function in Q28.

Solution:

$L=1$. $a_0 = \frac{1}{1}\int_{-1}^1 e^x dx = [e^x]_{-1}^1 = e^1 - e^{-1} = 2\sinh(1)$.

Question 30:

What is the weight function $r(x)$ for Bessel's equation $x^2 y'' + xy' + (\lambda^2 x^2 - \nu^2)y = 0$ written in S-L form?

Solution:

Divide by $x^2$: $y'' + \frac{1}{x}y' + (\lambda^2 - \frac{\nu^2}{x^2})y = 0$.

Integrating factor for first two terms: $e^{\int (1/x) dx} = e^{\ln x} = x$. This is $p(x)$.

Multiply original equation by $p(x)/A = x/x^2 = 1/x$:

$xy'' + y' + (\lambda^2 x - \frac{\nu^2}{x})y = 0$.

Recognize the first two terms as $(xy')'$.

$(xy')' - \frac{\nu^2}{x} y + \lambda^2 x y = 0$.

This matches the form $(py')' + qy + \lambda' r y = 0$ if we let $\lambda'$ represent the eigenvalue parameter $\lambda^2$.

So, $p(x)=x$, $q(x)=-\nu^2/x$, and the weight function is $r(x)=x$.
